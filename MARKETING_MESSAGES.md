## 버전 1 — 짧고 임팩트 있게 (30초 읽기)

**사용 상황**: 일반 지인, 빠르게 소개할 때

```
안녕하세요! 제가 만들고 있는 앱 잠깐 소개해도 될까요? 😊

📱 DreamPath — AI 진로 설계 앱

중고등학생이 "의사 되고 싶어요" 클릭 한 번으로
→ 중학교부터 대학까지 커리어 패스가 즉시 생성되고
→ 같은 꿈 가진 친구들과 런치패드 모임에서 함께 실행하는 앱이에요.

기존 300만원짜리 입시 컨설팅을 월 9,900원으로 대체합니다.

📊 시장: 진로진학 컨설팅 1,007억원 (매년 33% 성장 중)
👥 타겟: 중고등학생 260만명 + 학부모

현재 앱 MVP 완성 단계이고,
200개 직업 커리어 패스 DB 제작 중입니다.

혹시 중고등학생 자녀 있는 분 주변에 계시면 소개해 주실 수 있을까요? 🙏
```

## 버전 2 — 페르소나별 소개

### 학부모 지인에게

**사용 상황**: 중고등학생 자녀가 있는 학부모

```
안녕하세요! 제가 만드는 앱인데 한번 봐주실 수 있을까요?

아이가 "나 뭐 하고 싶은지 모르겠어요" 할 때,
입시 컨설팅 300만원 안 쓰고도
→ 200개 직업 중 클릭 한 번 → 커리어 패스 즉시 생성
→ 같은 꿈 가진 친구들과 모임에서 함께 실행

월 9,900원으로 해결하는 앱이에요.

강남 학생만 받던 입시 전략을
지방 학생도 똑같이 쓸 수 있게 만들고 있습니다.

테스트 해보실 분 있으면 연락 주세요!
```

---

### 교사/교육 관련 지인에게

**사용 상황**: 진로 교사, 진학 상담 교사, 교육 관련 종사자

```
진로 수업 16차시 채우기 힘드시죠?

DreamPath는 진로 교사가 수업 도구로 쓸 수 있게 만들었어요.
→ 학생들이 200개 직업 중 클릭 → 즉시 커리어 패스 생성
→ 학급 단위 런치패드 모임 → 팀 프로젝트 자동 구성
→ 교사 대시보드에서 학생 활동 기록 자동 저장 (생기부 근거 자료!)

학교 도입 파트너 찾고 있는데 관심 있으시면 말씀해 주세요 🙏
```

### 스타트업/투자 관련 지인에게

**사용 상황**: 투자자, 액셀러레이터, 스타트업 관계자

```
안녕하세요! 제가 에듀테크 스타트업 만들고 있는데 소개해도 될까요?

DreamPath — 중고등학생 AI 진로 설계 플랫폼

핵심 차별점:
① 클릭 한 번으로 커리어 패스 즉시 생성 (시장 유일)
② 런치패드 — 같은 꿈 동료끼리 모임 자동 매칭 (시장 유일)
③ 기존 300만원 컨설팅 → 월 9,900원 가격 파괴

시장: SAM 1,007억원 / YoY +33%
현재: MVP 완성, 200개 직업 패스 DB 제작 중
목표: Pre-Seed 2억원 유치

투자자 혹은 관심 있는 분 계시면 연결해 주실 수 있을까요?
```

## 버전 3 — 단계별 기능 설명 (포트폴리오용)

**사용 상황**: 상세 설명이 필요한 경우, 포트폴리오 공유, 페르소나 군집 설명

```
📱 DreamPath 소개

━━━━━━━━━━━━━━━━━━━━━━━━
🎯 누구를 위한 앱인가요?
━━━━━━━━━━━━━━━━━━━━━━━━

① 자유학기제 중학생 (김서연, 14세)
   "나한테 맞는 직업이 뭔지 모르겠다"
   → 수의사 클릭 → 커리어 패스 즉시 생성
   → 학급 런치패드 모임에서 동물 관련 직업 친구들과 팀 프로젝트

② 입시 준비 고등학생 (이준호, 16세)
   "생기부에 뭘 넣어야 하지?"
   → 의사 패스 클릭 → 과학 탐구 특화 계획으로 좁히기
   → 300만원 컨설팅 대신 학교 수업에서 바로 시작

③ 유학 준비 학생 (박하은, 17세)
   "유학원 1,000만원이 너무 비싸다"
   → 해외 대학 패스 → AP/SAT 로드맵 즉시 확인
   → 런치패드에서 CS 포트폴리오 함께 제작

④ 불안한 학부모 (최미영, 44세)
   "아이 진로를 어떻게 도와줘야 할지 모르겠다"
   → 학교 수업에서 시작한 걸 집에서도 계속 하는 모습
   → 월 2만원 × 2명 = 4만원으로 300만원 대체

━━━━━━━━━━━━━━━━━━━━━━━━
⚙️ 단계별 기능
━━━━━━━━━━━━━━━━━━━━━━━━

[1단계] 커리어 패스 설계 (메인)
• 200개 직업 중 클릭 한 번 → 즉시 내 커리어 패스 생성
• 중학교 ~ 대학까지 로드맵 자동 생성
• 나만의 범위·주제로 점점 좁혀가기
  (의사 → 소아과 전문의 → 희귀질환 전문)

[2단계] 런치패드 — 함께 실행 (활성화)
• 같은 커리어 패스 선택한 학생끼리 자동 매칭
• 세미나 모임 (현직자 강연)
• 프로젝트 모임 (3~5명 팀 빌딩)
• 스터디 모임 (주 1회 정기 학습)

[3단계] AI 멘토링 (서브)
• 탐구 보고서 구조 잡기
• 프로젝트 WBS 관리
• 공모전 준비 가이드

━━━━━━━━━━━━━━━━━━━━━━━━
💰 수익 모델
━━━━━━━━━━━━━━━━━━━━━━━━

• 무료: 200개 패스 탐색 + 기본 사용
• Explorer 월 9,900원: 나만의 패스 무제한 + AI 멘토
• Pioneer 월 19,900원: 커스텀 패스 제작·판매 + 프리미엄 모임

━━━━━━━━━━━━━━━━━━━━━━━━
📊 시장 기회
━━━━━━━━━━━━━━━━━━━━━━━━

• 사교육 시장 전체: 29.2조원
• 직접 타겟 시장: 1,007억원 (YoY +33%)
• 경쟁사 없음: 클릭으로 커리어 패스 즉시 생성하는 서비스 전무

현재 MVP 완성, 200개 직업 패스 제작 중입니다.
테스트해 보실 분 또는 소개해 주실 분 계시면 연락 주세요! 🙏
```

## 특별 버전 — 입시 컨설턴트 지인에게

**사용 상황**: 실제 입시 컨설팅 경험이 있는 전문가에게 피드백 요청

```
정선이 누나, 잘 지냈어? 😊

나 요즘 앱 하나 만들고 있는데, 누나한테 꼭 한번 보여주고 싶어서 연락했어.

입시 컨설팅 진로 설계를 앱으로 만드는 건데,
누나가 실제로 학생들 상담할 때 쓰는 것들을
이렇게 자동화하는 거야:

→ 학생이 200개 직업 중 클릭 한 번
   → 즉시 커리어 패스 로드맵 생성
→ 같은 목표 학생끼리 런치패드 모임 자동 매칭
   → 팀 프로젝트 함께 실행
→ 교사 대시보드에서 활동 기록 자동 저장
   → 생기부 근거 자료까지!

기존에 300만원짜리 입시 컨설팅을
월 9,900원으로 대체하는 게 목표야.

근데 솔직히 실제 컨설팅 현장이 어떻게 돌아가는지,
누나처럼 직접 해본 사람 시각이 꼭 필요해.

커리어 패스 로드맵 내용이 실제랑 맞는지,
어디서 수정해야 할지 같이 보면서 얘기할 수 있을까?

밥 한 끼 살게! 시간 될 때 한번 만나자 ☕
```

### 포인트 설명

| 요소 | 이유 |
|------|------|
| "누나한테 꼭 보여주고 싶어서" | 부탁이 아니라 신뢰 표현으로 시작 |
| 기능 3줄 요약 | 컨설턴트라 바로 이해 가능한 언어로 |
| "직접 해본 사람 시각이 꼭 필요해" | 전문가로서 존중 + 만남의 명분 자연스럽게 |
| "밥 한 끼 살게" | 부담 없이 YES 하기 쉬운 마무리 |

## 상황별 선택 가이드

| 지인 유형 | 추천 버전 | 핵심 메시지 |

## 추가 활용 팁

### 카톡 메시지 전송 전 체크리스트

- [ ] 상대방의 관심사·직업에 맞는 버전 선택
- [ ] 이름이나 호칭 개인화 (예: "정선이 누나")
- [ ] 구체적인 부탁 포함 (테스트, 소개, 만남 등)
- [ ] 부담 없는 마무리 (밥 한 끼, 커피 한 잔 등)

### 후속 메시지 예시

**긍정 반응 시:**
```
오! 관심 가져줘서 고마워 😊
간단한 데모 영상 보내줄게.
[데모 영상 링크]

다음 주 중에 시간 괜찮아?
```

**질문이 들어올 시:**
```
좋은 질문이야!
[구체적 답변]

더 자세한 건 만나서 얘기하는 게 나을 것 같아.
언제 시간 괜찮아?
```
