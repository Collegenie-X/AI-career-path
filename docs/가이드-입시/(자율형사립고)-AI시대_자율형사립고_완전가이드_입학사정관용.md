## 1. 문서 개요 및 활용 안내

### 1.1 이 문서는 누구를 위한 것인가

본 문서는 **고등학교 진학을 앞둔 중학생과 학부모를 상담하는 입학사정관**을 위한 자율형사립고(자사고) 종합 레퍼런스입니다. 2026년 기준 전국 33개교(전국단위 10개교, 서울 광역단위 15개교, 비서울 광역단위 8개교)의 학교별 특성, 입시 전략, 2028 대입 개편 대응 전략, AI 시대 생존 전략을 체계적으로 정리했습니다.

## 2. AI 시대 교육 패러다임의 전환

### 2.1 AI가 바꾸는 엘리트 교육

AI(ChatGPT, Claude 등)의 등장으로 **엘리트 교육의 정의가 근본적으로 변화**하고 있습니다. 과거 자사고가 독점하던 '심화 선택과목 우위'는 2028 통합형 수능으로 사라지며, 결국 **'무엇을 알고 있는가'가 아니라 '무엇을 할 수 있는가'**가 변별력이 됩니다.

| 구분 | 과거 엘리트 교육 | AI 시대 엘리트 교육 |
|------|-----------------|-------------------|
| **핵심 역량** | 빠른 문제 풀이, 지식 암기량 | 문제 정의, 비판적 사고, 메타인지 |
| **평가 방식** | 정답 찾기 (객관식 + 단답형) | 과정 평가 (세특 + 면접 + 서술형) |
| **학습 방식** | 학원 → 선행 → 반복 풀이 | 자기주도 탐구 → R&E → AI 도구 활용 |
| **AI와의 관계** | AI 미존재 (학원 의존) | AI를 도구로 활용하되 해석은 본인 |
| **자사고의 가치** | 심화 선택과목 + 내신 경쟁 | **세특 깊이 + 면접 논리 + R&E 경험** |
| **변별력 원천** | 선행 범위, 풀이 속도 | '왜 했는지·무엇을 발견했는지' 설명 능력 |

> **상담 포인트**: "AI 시대에 자사고의 가치는 '심화 문제를 더 많이 풀 수 있는 환경'이 아니라, **'자기만의 탐구를 깊이 있게 할 수 있는 환경'**에 있습니다. 학원에서 떠먹여 줘야 공부하는 학생은 자사고에서 오히려 도태됩니다."

### 2.3 학종형 vs 정시형 DNA의 의미

## 3. 자율형사립고 제도 개요

### 3.1 자사고란 — 법적 근거

자율형사립고(자사고)는 **초·중등교육법 시행령 제91조의3**에 근거한 고등학교 유형입니다.

| 항목 | 내용 |
|------|------|
| **법적 근거** | 초·중등교육법 시행령 제91조의3 |
| **정의** | 교육과정 편성·운영의 자율성이 보장된 사립 고등학교 |
| **지정 주체** | 교육부장관(전국단위) 또는 시·도 교육감(광역단위) |
| **지정 기간** | 5년 (재지정 심사를 통해 갱신) |
| **학비** | 일반고 등록금의 3배 이내 (실제로는 학교별 상이) |
| **교육과정 자율성** | 국가 교육과정의 50% 이상 자율 편성 가능 |
| **교사 채용** | 독자적 교사 채용 가능 (전문 교과 교사 초빙) |
| **2026년 현재 학교 수** | 전국 33개교 (전국단위 10 + 서울 광역 15 + 비서울 광역 8) |

### 3.2 전국단위 vs 광역단위 비교

## 8. 2028 대입 개편과 자사고 전략

### 8.1 내신 5등급제 — 자사고에 미치는 영향

| 구분 | 현행 (9등급 상대평가) | 2028 (5등급 절대평가) |
|------|---------------------|---------------------|
| **등급 체계** | 1~9등급 (상대평가) | A~E등급 (절대평가) |
| **1등급 비율** | 상위 4% | A등급 기준 점수 이상 (비율 무관) |
| **자사고 영향** | 내신 불리 (최상위끼리 경쟁) | **중립적** — A등급 취득 용이해지나 변별력 감소 |
| **학종 변별력** | 내신 등급으로 변별 | **세특·면접으로 변별** |
| **정시 영향** | 내신과 무관 | 내신과 무관 |

**A등급의 의미 분석**:

| 학교 유형 | A등급 취득 난이도 | A등급 비율(추정) | 변별력 |
|----------|-----------------|----------------|--------|
| 전국단위 자사고 | 높음 (최상위 집결) | 40~60% | **세특으로 차별화 필수** |
| 서울 광역 자사고 | 중상 | 50~70% | 세특+면접으로 차별화 |
| 일반고 | 중 | 30~50% | 내신 등급 자체로 차별화 어려움 |

> **상담 포인트**: "5등급제에서 자사고 학생은 A등급 취득은 어렵지만, **A등급 안에서의 변별력은 세특으로만 가능**합니다. 이것이 자사고 학종형 DNA의 핵심 가치입니다."

### 8.2 학종형 자사고 전략 (세특 35~40%, R&E 활용)

| 전략 항목 | 세부 내용 | 실행 시기 |
|----------|----------|----------|
| **세특 강화** | 전 과목 탐구 활동 기록 (교과서 넘어선 심화 주제) | 고1~고2 |
| **R&E 프로젝트** | 대학교수 지도 하 1년 장기 연구 (세특 핵심 콘텐츠) | 고1 하반기~고2 |
| **비교과 활동** | 동아리·봉사·자율활동 (단, AI로 차별화 어려운 활동 피하기) | 고1~고3 |
| **면접 준비** | 생기부 기반 심층 질문 대비 (예상 질문 100개 리스트) | 고3 상반기 |
| **수능 최저** | 학종에 필요한 수능 최저 충족 (국·수·영·탐 2개 합 5~6) | 고2~고3 |

### 8.3 정시형 자사고 전략 (수능 킬러문항 대비)

| 전략 항목 | 세부 내용 | 실행 시기 |
|----------|----------|----------|
| **수능 집중** | 국·수·영·탐 4과목 최적화 (수학 킬러문항 정복) | 고1~고3 |
| **통합형 수능** | 2028 통합형 수능 대비 — 공통+선택 구조 변화 파악 | 고1부터 |
| **모의고사** | 월 2회 이상 모의고사 + 오답 분석 | 고2~고3 |
| **EBS 연계** | EBS 연계 교재 완벽 소화 | 고3 |
| **의대 정시** | 의대 정시 커트라인 분석 + 수능 점수 목표 설정 | 고2~고3 |

### 8.4 교과전형 SKY 진입 전략

| 전략 | 대학 | 전형명 | 핵심 요건 | 자사고 유불리 |
|------|------|--------|----------|-------------|
| 교과전형 | 서울대 | 지역균형선발 | 학교장 추천 + 내신 + 수능최저 | **불리** (내신 경쟁 치열) |
| 교과전형 | 연세대 | 추천형 | 학교장 추천 + 교과 | **불리** (내신 등급) |
| 교과전형 | 고려대 | 학교추천 | 학교장 추천 + 교과 | **불리** (내신 등급) |

> **상담 포인트**: "교과전형에서 자사고는 **불리**합니다. 내신 경쟁이 치열해 1등급 확보가 어렵기 때문입니다. 자사고 학생은 **학종 또는 정시**를 주 전략으로 삼아야 합니다."

### 8.5 의대 진학 전략 (학종 의대 vs 정시 의대)

| 구분 | 학종 의대 | 정시 의대 |
|------|---------|---------|
| **적합 학교** | 하나고, 외대부고 (학종형) | **상산고, 현대청운고 (정시형)** |
| **핵심 준비** | 세특(생명과학·화학 심화) + R&E + 봉사 | 수능 전 과목 최상위 (수학 만점 목표) |
| **장점** | 의대 학종은 내신 불리를 세특으로 보완 가능 | 의대 정시는 수능 점수만으로 결정 — 명확한 기준 |
| **단점** | 세특 변별력 어려워짐 (AI 시대) | 수능에서 실수 하나가 합불 결정 |
| **성공 확률** | 중상 (세특 깊이에 따라 결정) | **상** (정시형 자사고 인프라 활용 시) |
| **AI 시대 변화** | AI 활용 R&E가 새로운 차별화 요소 | AI가 문제풀이 보조 → 인간의 응용력이 관건 |

### 8.6 해외 대학 병행 전략 (민사고·하나고·외대부고)

| 학교 | 해외 대학 강점 | 병행 전략 | 적합 학생 |
|------|--------------|----------|----------|
| **민사고** | 아이비리그·옥스브리지 | AP/SAT + 국내 수시 병행 | 영어 최상위 + 글로벌 리더십 |
| **하나고** | 미국 Top 30 | R&E 기반 미국대 에세이 소재 확보 | 자기주도형 + 연구 경험 |
| **외대부고** | 미국·영국 명문대 | 외국어 능력 + 국제 교류 경험 | 외국어 특기 + 국제 감각 |

## 10. AI 시대 자사고 생존 전략

### 10.1 AI가 대체하는 것 vs 인간이 해야 할 것

| AI가 대체하는 것 | 인간이 해야 할 것 |
|----------------|----------------|
| 문제 풀이 자동화 | **문제 정의 (무엇이 문제인가?)** |
| 개념 설명 및 요약 | **메타인지 (내가 무엇을 모르는가?)** |
| 학습 계획 제안 | **자기주도 학습 전략 수립** |
| 오답 분석 | **오답의 근본 원인 진단** |
| 자료 수집·번역 | **해석과 의미 부여** |
| 에세이 초안 작성 | **세특 스토리 구성 (개인 경험 연결)** |
| 면접 예상 질문 생성 | **면접에서의 즉흥 논리 전개** |

| **핵심 답변** | 2024년 1월 교육부가 시행령을 개정하여 자사고·외고·국제고 존치를 확정했습니다. 그러나 시행령은 대통령령이므로 정권 교체 시 재개정이 가능합니다. **완전한 보장은 없지만, 현실적으로 입학 시점부터 3년간은 유지될 가능성이 높습니다.** |
| **근거·데이터** | 2019년 폐지 결정 → 2024년 존치 확정까지 5년 소요. 정권 교체 후에도 시행령 개정에 최소 1~2년 소요. 현재 재학생에 대한 경과조치는 법적으로 보장됨 |
| **실전 전략** | 1) 입학 시점 기준 3년 유지는 현실적으로 보장. 2) 그 이후는 정치 상황에 따라 변동 가능. 3) 자사고가 폐지되더라도 재학생 경과조치 보장. 4) 불확실성이 너무 불안하다면 일반고를 선택하는 것도 합리적 판단 |

**Q2. "학비가 연 800~2,000만원이라고 하는데, 이 비용이 정확히 어디에 쓰이나요?"**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | 자사고 학비는 크게 **등록금(수업료) + 기숙사비 + 급식비 + 방과후 교육비 + 교재비**로 구성됩니다. 일반고(연 100만원 내외)의 3~20배 수준이며, 그 차이는 주로 소규모 학급 운영, 전문 교사 초빙, 기숙사 시설, R&E 프로그램 운영에 투입됩니다. |
| **비용 구조 분석** | 전국단위: 등록금 30~40% + 기숙사·급식 30% + 방과후·교재 20% + 시설·운영 10%. 광역단위: 등록금 50~60% + 방과후·교재 20~30% + 시설·운영 10~20% |
| **핵심 질문** | "이 비용 대비 어떤 교육적 가치를 얻는가?"를 학교 설명회에서 반드시 질문하세요. 학교알리미(schoolinfo.go.kr)에서 1인당 교육비 공시 정보 확인 가능 |

**Q3. "대광고처럼 자사고가 자발적으로 일반고로 전환하는 경우, 재학 중인 학생은 어떻게 되나요?"**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | 대광고는 2026학년도 **신입생부터** 일반고로 운영합니다. 기존 재학생(2024·2025 입학생)은 **자사고 교육과정을 그대로 이수**합니다. 법적으로 재학생에 대한 경과조치가 보장됩니다. |
| **근거·데이터** | 대광고는 서울에서 12번째로 자사고를 자발적으로 반납한 사례(메트로서울 2025.8 보도). 학교법인의 판단에 따른 반납이며, 재지정 심사 탈락은 아님 |
| **의사결정 가이드** | 1) 입학 전 해당 학교의 재지정 심사 일정 확인 (5년 단위). 2) 학교법인의 장기 운영 의지 확인 (설명회에서 질문). 3) 재학 중 전환되더라도 경과조치 보장 → 교육 공백은 없음 |

**Q4. "자사고에 대한 교육 감독 체계는 어떻게 되나요? 학교가 학비만 받고 교육은 부실하게 하면요?"**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | 자사고는 **5년마다 재지정 심사**를 받습니다. 교육과정 운영 실적, 학생 선발 적정성, 교육 성과, 운영 건전성을 평가하며, 미충족 시 일반고로 전환됩니다. 또한 학교알리미를 통해 교육비·성과 정보가 공개됩니다. |
| **근거·데이터** | 교육부·교육청 합동 평가 + 학교알리미 공시 + 학부모 만족도 조사. 2019~2025년 사이 다수의 자사고가 실적 부진으로 자발적 전환 |
| **실전 전략** | 학교알리미(schoolinfo.go.kr)에서 해당 학교의 1인당 교육비, 교사 1인당 학생 수, 진학 실적을 직접 확인하세요 |

**Q5. "전국단위 자사고와 광역단위 자사고의 법적·제도적 차이가 정확히 무엇인가요?"**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | 전국단위는 **교육부장관이 지정**하고 전국에서 학생을 모집합니다. 광역단위는 **시·도 교육감이 지정**하고 해당 시·도 내에서만 모집합니다. 전국단위는 기숙사가 의무이고 학비가 높으며, 광역단위는 통학이 기본이고 학비가 상대적으로 낮습니다. |
| **핵심 차이표** | 3.2절 비교표 참조. 핵심: 지정 주체(교육부 vs 교육감), 모집 범위(전국 vs 시도), 기숙사(의무 vs 선택), 학비(높음 vs 중간), 학생 수준(전국 최상위 vs 지역 상위) |
| **의사결정 가이드** | 전국 최상위를 목표로 하면 전국단위. 통학 편의 + 지역 내 우수 환경을 원하면 광역단위. "같은 자사고도 다른 학교" — 반드시 구분하여 상담 |

#### B. 학교 DNA·성과 (Q6~Q10)

**Q6. "학종형 DNA와 정시형 DNA는 어떻게 형성되는 건가요? 학교가 의도적으로 만드는 건가요?"**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | DNA는 학교의 **설립 이념 + 교육과정 편성 + 교사 구성 + 졸업생 실적 패턴**이 축적되어 형성됩니다. 의도적 설계와 자연스러운 축적이 모두 작용합니다. |
| **근거·데이터** | 하나고: 설립 이념이 '자기주도 학습' → 교육과정이 탐구·토론 중심 → 학종 실적 축적 → 학종형 DNA 확립. 상산고: 설립 초기부터 수능 집중 인프라 구축 → 의대 정시 실적 축적 → 정시형 DNA 확립 |
| **실전 전략** | DNA는 10년 이상의 축적 결과. 최근 1~2년 실적만 보지 말고 **5~10년 추이**를 확인하세요. DNA가 변하는 학교(예: 정시형 → 학종형 전환 시도)는 과도기 리스크가 있음 |

**Q7. "R&E 프로그램이 실제로 대입에 얼마나 도움이 되나요?"**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | R&E는 학종에서 **가장 강력한 세특 소재**입니다. 대학교수 지도 하 1년 장기 연구 경험은 일반고에서 쉽게 접하기 어려운 자사고만의 장점입니다. 서울대 자연계 학종 합격생의 상당수가 R&E 경험을 보유하고 있습니다. |
| **근거·데이터** | R&E 효과: 세특 기재 시 탐구 깊이 차별화, 면접에서 연구 과정 설명 가능, 학문적 호기심 입증. 단, R&E 자체가 합격을 보장하지는 않음 — **연구 과정의 진정성**이 핵심 |
| **실전 전략** | R&E 참여 시: 1) 주제를 본인이 선정 (교수 제안을 그대로 따르지 않기). 2) 과정을 세특에 구체적으로 기록. 3) 면접에서 "AI가 못한 것, 내가 한 것"을 설명할 수 있어야 함. AI 시대에는 R&E에서 AI를 도구로 어떻게 활용했는지가 새로운 평가 포인트 |

**Q8. "자사고 SKY 진학률이 70~80%라는 말이 사실인가요?"**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | **학교별 편차가 매우 큽니다.** 전국단위 최상위(상산고 등)는 SKY 50~70%까지 가능하지만, 광역단위 자사고 중에는 SKY 10~20%에 그치는 곳도 있습니다. "자사고 = SKY 70%"는 일부 학교의 실적을 전체에 일반화한 것입니다. |
| **근거·데이터** | 전국단위 상위: SKY 40~70%. 전국단위 하위: SKY 25~35%. 서울 광역 상위(세화·중동·휘문): SKY 20~40%. 서울 광역 하위: SKY 10~20%. 비서울 광역: 학교별 편차 큼 |
| **실전 전략** | 1) 학교 공식 발표 실적을 확인하되, "중복 합격"과 "최종 등록" 구분 필요. 2) 최근 3년 추이를 확인. 3) "SKY 진학률"보다 **"나와 비슷한 성적대의 선배가 어디에 갔는가"**가 더 유의미한 지표 |

**Q9. "자사고에서 중도 탈락(자퇴)하는 학생의 유형과 비율은?"**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | 전국단위 기숙사 학교에서 중도탈락률은 **연간 1~5%** 수준으로 추정됩니다. 주요 원인은 1) 기숙사 생활 부적응, 2) 성적 하락에 따른 번아웃, 3) 교우관계 갈등, 4) 진로 변경(해외 유학 등)입니다. |
| **근거·데이터** | 탈락 유형: 1학기 적응 실패(가장 많음) > 2학년 번아웃 > 진로 변경. 광역단위 통학 학교는 탈락률 낮음 |
| **실전 전략** | 1) 입학 전 기숙사 체험 프로그램 참여. 2) 학교 내 상담·멘토링 체계 확인. 3) "적응 못 하면 전학 가능한가?" 사전 확인. 4) 학부모: 자녀의 독립심·자기관리 능력을 객관적으로 평가 |

**Q10. "서울 광역단위 자사고 15개교 사이에도 격차가 큰가요?"**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | **매우 큽니다.** 서초·강남권(세화고, 중동고, 휘문고, 현대고)과 기타 지역 자사고 사이에 대입 실적 차이가 있습니다. 같은 "서울 자사고"라는 이름으로 묶이지만, 실제 교육 환경·학생 수준·대입 실적은 학교별로 상당히 다릅니다. |
| **근거·데이터** | 강남·서초권: 학원 밀집 + 학부모 교육열 → 학생 수준 상대적 우위. 강북·도봉권: 지역 내 최상위이나 강남권 대비 실적 격차 존재 |
| **의사결정 가이드** | 1) 거주지 기준으로 통학 가능한 학교 먼저 선별. 2) 해당 학교의 최근 3년 SKY 진학 실적 직접 확인. 3) 학교 설명회에서 "우리 학교 SKY 진학 실적"을 정확히 질문. 4) "서울 자사고니까 다 비슷하겠지"는 위험한 가정 |

#### C. 2028 대입 전략 (Q11~Q15)

**Q11. "2028 내신 5등급 절대평가가 자사고에 어떤 영향을 미치나요? A등급이면 변별력이 없는 거 아닌가요?"**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | 5등급제는 자사고에 **중립적**입니다. A등급 취득이 쉬워지는 것은 사실이지만, 자사고 내부 모집단 자체가 최상위이므로 A등급 내에서의 변별력은 **세특**에 의해 결정됩니다. 즉, 내신이 아니라 세특이 당락을 가릅니다. |
| **근거·데이터** | 자사고 A등급 비율 추정: 40~60%. 일반고 A등급 비율 추정: 30~50%. A등급 안에서의 차이는 세특 깊이로 변별. 2028 학종에서 세특 비중 35~40%로 확대 전망 |
| **실전 전략** | 1) A등급 확보는 기본 — 내신 관리 소홀 금지. 2) **세특에 시간·에너지 집중 투자**. 3) R&E·탐구활동의 깊이가 합격과 불합격을 가르는 핵심. 4) "A등급이니까 내신 걱정 없다"는 착각 금지 — A 안에서도 줄 세우기는 존재 |

**Q12. "2028 대입에서 학종형 자사고와 정시형 자사고 중 어디가 유리한가요?"**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | **학종형이 상대적으로 유리**해지는 추세입니다. 5등급제로 내신 변별력이 감소하면 세특·면접이 핵심이 되고, 이는 학종형 자사고의 강점입니다. 정시형은 통합형 수능에서 기존의 '심화 선택과목 우위'가 사라지므로 전략 수정이 필요합니다. |
| **근거·데이터** | 학종형 자사고: 세특 인프라 → 2028 학종에서 강점 유지. 정시형 자사고: 수능 인프라는 여전히 강하지만, 통합형 수능으로 차별화 포인트 변경 |
| **실전 전략** | 학종형: 기존 전략 유지 + AI 활용 세특 강화. 정시형: 수능 집중 유지 + 세특 보완 (학종 일부 병행). 균형형: 수시 60% + 정시 40% 유지 |

**Q13. "의대 합격을 위해 자사고에서 학종과 정시 중 어떤 전략이 좋나요?"**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | **의대 정시: 상산고·현대청운고가 최적.** 수능 전 과목 최상위 인프라가 구축되어 있습니다. **의대 학종: 하나고·외대부고에서 가능.** 세특(생명과학·화학 심화) + R&E + 봉사가 핵심입니다. 양쪽 모두 의대 합격 사례가 있으므로 학생의 DNA에 따라 선택하세요. |
| **근거·데이터** | 의대 정시: 수능 수학 만점 + 과탐 만점급 필요. 상산고 정시 의대 합격 실적 전국 최상위. 의대 학종: 세특 깊이 + 면접(MMI) 대비. 하나고 학종 의대 합격 사례 다수 |
| **실전 전략** | 1) 수능형 학생 → 상산고 정시 의대. 2) 탐구형 학생 → 하나고 학종 의대. 3) 양방향 → 균형형 자사고에서 수시+정시 병행. 의대 정원 확대 추세에서 자사고는 학종+정시 양쪽 모두 강력한 경쟁력 유지 |

**Q14. "서울대 지역균형전형에서 학교장 추천은 자사고에 유리한가요?"**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | 서울대 지역균형선발전형은 학교당 추천 인원이 **제한**됩니다 (보통 2~3명). 자사고에서는 내부 경쟁이 치열하여 추천 받기 자체가 어렵습니다. 반면 일반고에서는 추천 경쟁이 상대적으로 낮아 유리할 수 있습니다. |

| **실전 전략** | 서울대 지균을 주 전략으로 삼는다면 일반고가 유리. 서울대 일반전형(학종)은 자사고 세특 강점 활용 가능. 전략에 따라 학교 선택이 달라짐 |

**Q15. "해외 대학 병행 전략이 현실적으로 가능한가요? 수능도 준비하면서?"**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | **민사고·하나고·외대부고**에서는 현실적으로 가능합니다. 이들 학교는 AP/SAT 준비, 해외 대학 원서 작성, 추천서 등의 인프라를 보유하고 있습니다. 단, 국내 수능과 해외 대학 준비를 동시에 하면 부담이 크므로, 고2까지 방향을 결정하는 것이 좋습니다. |

#### D. AI 시대 경쟁력 (Q16~Q20)

**Q16. "AI 시대에 심화 문제풀이 중심 교육이 여전히 가치가 있나요?"**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | **문제풀이 자체의 가치는 감소하지만, 문제풀이 과정에서 훈련되는 논리적 사고력은 여전히 가치 있습니다.** AI가 문제를 풀어주는 시대에, 중요한 것은 '어떤 문제를 풀어야 하는가'를 정의하는 능력입니다. 정시형 자사고도 이 점을 인식하고 교육 방법을 진화시킬 필요가 있습니다. |
| **근거·데이터** | AI(GPT-4 이상)는 수능 수학 킬러문항의 상당수를 풀 수 있음. 그러나 수능은 여전히 '인간이 제한 시간 내 풀기'를 요구 → 당분간 문제풀이 훈련의 입시적 가치는 유지. 장기적으로는 서술형·과정형 평가로 전환 예상 |
| **실전 전략** | 1) 수능 준비는 여전히 필요 (정시 전략). 2) 문제풀이 + 탐구 능력의 균형이 중요. 3) "문제를 푸는 기계"가 아니라 "문제를 정의하는 인간"으로 성장 |

**Q17. "자사고에서 AI 도구를 활용한 학습이 허용되나요? 세특에 기재할 수 있나요?"**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | **허용되며, 올바르게 활용하면 세특에서 차별화 요소**가 됩니다. 핵심은 'AI에게 답을 받은 것'이 아니라 'AI를 도구로 활용하되 해석과 결론은 본인이 도출한 과정'을 기록하는 것입니다. |
| **근거·데이터** | 2025~2026 학종 평가 트렌드: "AI 활용 여부"보다 **"AI를 어떻게 활용했고, AI가 못한 부분을 본인이 어떻게 보완했는가"**를 평가. 세특에 "ChatGPT로 자료를 수집한 후, 3가지 오류를 발견하여 직접 원문 확인"식 기술이 차별화 |
| **실전 전략** | 10.3절 AI 활용 4대 원칙 참조. 세특 기재 시: "AI 활용 과정 + AI 한계 발견 + 본인 보완"의 3단계 구조로 서술 |

**Q18. "자사고 동문 네트워크가 AI 시대에도 가치가 있나요?"**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | **AI 시대에 동문 네트워크의 가치는 오히려 증가합니다.** AI가 정보를 평준화하는 시대에, '누구와 연결되어 있는가'라는 사회적 자본은 AI가 대체할 수 없는 인간 고유의 자산입니다. 자사고 졸업생 네트워크는 대학 진학·취업·창업에서 지속적 가치를 제공합니다. |
| **근거·데이터** | 전국단위 자사고(특히 하나고·민사고·상산고)의 동문 네트워크는 의사·법조인·교수·대기업 임원 등 사회 지도층에 분포. 대학 입학 후 선배 멘토링, 취업 시 추천, 창업 시 투자 연결 등의 실질적 가치 |
| **실전 전략** | 1) 동문 네트워크는 **부가 가치**이지 **입학 이유**가 되어서는 안 됨. 2) 네트워크의 질은 학교 역사·졸업생 수에 비례. 3) 오래된 학교(배재고 1885년, 김천고 1931년)의 동문 네트워크가 두꺼움 |

**Q19. "자사고에서 기업가 정신·창업 교육을 받을 수 있나요?"**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | 대부분의 자사고는 기업가 정신·창업 교육을 **체계적으로 제공하지는 않습니다.** 자사고의 핵심은 대입 준비이며, 창업 교육은 대학에서 본격적으로 접하게 됩니다. 다만, R&E·프로젝트 기반 학습에서 기업가적 사고(문제 발견 → 해결책 설계)를 간접 훈련할 수 있습니다. |
| **실전 전략** | 1) 창업에 관심 있다면: 자사고보다 과학고·영재고의 창업 프로그램이 더 체계적일 수 있음. 2) 자사고 내에서: R&E를 사회적 문제 해결 주제로 설계하면 기업가적 사고 훈련 가능. 3) 동아리를 통한 자체적 프로젝트 운영 |

**Q20. "자사고 교육 모델의 구조적 한계는 무엇인가요? 솔직하게 말해주세요."**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | 솔직하게 말하면 자사고의 구조적 한계는 존재합니다. 1) **학비 장벽**: 경제적 불평등이 교육 기회의 불평등으로 이어짐. 2) **내신 불리**: 최상위끼리 경쟁하므로 교과전형에서 불리. 3) **정책 불안정성**: 정권 변화에 따른 존폐 리스크. 4) **획일적 엘리트 교육**: 다양성보다 대입 실적에 집중. 5) **AI 시대 적응 속도**: 전통적 교육 방식의 변화가 느릴 수 있음. |
| **근거·데이터** | 학비 상위 5개교 평균: 연 1,300만원. 이는 중위소득 가구의 연간 교육비 예산을 크게 초과. 내신 불리: 자사고 내 3등급이 일반고 1등급보다 실질 학력이 높을 수 있으나, 학종에서는 등급이 그대로 반영 |
| **실전 전략** | 이 한계를 인식한 상태에서 자사고를 선택하는 것과, 한계를 모른 채 선택하는 것은 다릅니다. **한계를 알고도 선택할 만한 이유가 있는지**를 학생·학부모와 함께 판단하세요 |

**Q22. "서울 광역단위 자사고는 일반고와 실질적으로 얼마나 다른가요? 솔직한 답변 부탁합니다."**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | **솔직히 말하면, 서울 광역단위 자사고의 일부는 일반고와 큰 차이가 없을 수 있습니다.** 차이는 주로 1) 학생 모집단 수준(자사고가 약간 높음), 2) 교육과정 자율성(자사고가 높음), 3) 학비(자사고가 3~5배)입니다. 이 차이가 학비 차이(연 600~800만원)만큼의 가치가 있는지는 **학교별로 다릅니다.** |
| **근거·데이터** | 서울 광역 자사고 SKY 진학률: 10~40% (학교별 편차 큼). 서울 상위 일반고 SKY 진학률: 5~15%. 차이가 크지 않은 학교도 있음 |
| **의사결정 가이드** | 1) 해당 학교의 SKY 진학 실적을 일반고와 비교. 2) 학비 차이 대비 교육적 가치를 평가. 3) "자사고라서 좋다"가 아니라 "이 학교가 우리 아이에게 좋다"가 판단 기준. 4) 강남·서초권 자사고와 기타 지역 자사고의 실적 격차 확인 필수 |

**Q23. "기숙사 생활이 아이의 정신건강에 부정적 영향을 줄 수 있지 않나요?"**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | **가능합니다.** 기숙사 생활은 독립심과 사회성을 키우는 장점이 있지만, 동시에 홈시크·교우관계 갈등·수면 부족·번아웃의 위험도 있습니다. 학교의 정신건강 지원 체계(상담사, 멘토링, 위기 대응)를 반드시 확인하세요. |
| **근거·데이터** | 전국단위 기숙사 학교: 1학기 적응 기간이 가장 위험. 적응 실패 시 학업·인간관계 동반 악화. 학교별 상담사 배치 현황 확인 필요 |
| **의사결정 가이드** | 1) 자녀의 독립심 수준 객관적 평가 (한 번도 혼자 잔 적 없다면 경고). 2) 학교 상담 체계 사전 확인. 3) 격주 외박·주말 귀가 정책 확인. 4) 적응 실패 시 전학 가능성 사전 확인. 5) "자사고에 보내는 것이 부모의 꿈인가, 아이의 꿈인가" 솔직한 자기 점검 |

**Q24. "자사고 불합격 리스크를 감수할 만한 가치가 있나요?"**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | **리스크-리턴 분석이 필요합니다.** 자사고 불합격 시 후기고(일반고) 배정을 받게 되며, 원하는 일반고에 배정되지 않을 수 있습니다. 합격 확률(경쟁률 기반)과 불합격 시 대안을 사전에 준비해야 합니다. |
| **의사결정 가이드** | 리스크: 불합격 → 원하지 않는 일반고 배정 가능. 리턴: 합격 시 자사고의 교육 환경·대입 인프라 활용. 판단 기준: 1) 경쟁률이 3:1 이하면 도전 가치 있음. 2) 경쟁률이 5:1 이상이면 불합격 대비 대안 필수. 3) 불합격 시 배정될 수 있는 일반고 목록 사전 확인. 4) "어떤 학교를 다니든 3년을 어떻게 보내느냐가 핵심"이라는 마인드셋 |

**Q25. "자사고가 교육 불평등을 심화시킨다는 비판에 대해 어떻게 생각하시나요?"**

| 항목 | 내용 |
|------|------|
| **핵심 답변** | **구조적 한계를 인정합니다.** 자사고의 높은 학비(연 800~1,900만원)는 중산층 이하 가정에게 진입 장벽입니다. 장학금 제도가 있지만, 모든 저소득 학생이 혜택을 받지는 못합니다. 자사고가 '부유한 학생을 위한 학교'라는 비판은 일정 부분 사실입니다. 그러나 하나고(하나금융 전액 장학)처럼 저소득 가정도 접근 가능한 학교가 있으며, 교육의 질 향상이 전체 사회에 기여하는 측면도 있습니다. |
| **실전 전략** | 1) 이 비판을 회피하지 말고 **솔직하게 인정**하세요. 2) 동시에, 장학금·재정지원 제도를 적극 안내. 3) "불평등을 심화시키지 않으려면 어떤 정책이 필요한가"에 대한 토론으로 전환. 4) 궁극적으로, **학생에게 최선의 교육 환경을 제공하는 것**이 입학사정관의 역할 |

### 11.6 실제 상담 시나리오

**시나리오 1: "우리 아이가 하나고에 가고 싶어하는데, 학비가 걱정입니다"**

| 단계 | 상담 내용 |
|------|----------|
| 1. 학생 적합성 확인 | 11.4 체크리스트로 적합성 진단 → 16점 이상이면 적극 추천 |
| 2. 학비 부담 분석 | 하나고 총 비용: 연 800~900만원 (동급 최저). 하나금융 장학재단 확인 |
| 3. 장학금 안내 | 소득 3분위 이하: 등록금+기숙사비 전액 지원 가능. 성적 우수 장학금도 별도 |
| 4. 최종 조언 | "하나고는 자사고 중 학비가 가장 낮고, 장학금 제도가 가장 잘 갖춰져 있습니다. 장학금 수혜 가능 여부를 먼저 확인하세요." |

**시나리오 2: "의대를 보내고 싶은데, 자사고 vs 일반고 중 어디가 좋을까요?"**

| 단계 | 상담 내용 |
|------|----------|
| 1. 학생 유형 파악 | 수능형(문제풀이 강점) vs 탐구형(세특 강점) |
| 2. 전략 분기 | 수능형 → 상산고(정시 의대 최강). 탐구형 → 하나고(학종 의대 가능). 양방향 → 균형형 자사고 |
| 3. 일반고 대안 | 일반고에서 내신 1등급 + 정시 집중도 유효한 전략 |
| 4. 최종 조언 | "의대 합격의 핵심은 학교가 아니라 학생의 실력과 전략입니다. 자사고의 인프라가 필요한지, 일반고의 내신 유리함이 필요한지를 판단하세요." |

**시나리오 3: "자사고에 보냈는데 성적이 떨어져서 고민입니다"**

| 단계 | 상담 내용 |
|------|----------|
| 1. 현황 파악 | 성적 하락 원인: 적응 실패? 학습 방법 문제? 정신건강 이슈? |
| 2. 해결 방안 | 학교 내 상담사 연결, 학습 멘토링 신청, 자기주도 학습 계획 재수립 |
| 3. 최악의 경우 | 일반고 전학도 옵션 — "자사고 자퇴"는 실패가 아님 |
| 4. 최종 조언 | "자사고에서 3등급이 일반고 1등급보다 실질 학력이 높을 수 있습니다. 하지만 대입에서는 등급이 중요하므로, 학종 전략을 재설계하거나 정시에 집중하는 것을 검토하세요." |

## 부록 B: 자소서 작성 가이드 (실전 예시)

### 자소서 구조 (STAR 기법 적용)

| 단계 | 영문 | 설명 | 예시 |
|------|------|------|------|
| **S** | Situation | 상황 제시 | "중2 수학 시간, 이차방정식 판별식 단원에서 3회 연속 오답" |
| **T** | Task | 과제/목표 | "판별식의 기하학적 의미를 스스로 이해하겠다는 목표 설정" |
| **A** | Action | 행동 | "교과서 → 참고서 → AI(ChatGPT)에 질문 → AI 답변 검증 → 직접 그래프 작성" |
| **R** | Result | 결과/성장 | "판별식과 포물선의 관계를 이해. '모르는 것을 정확히 찾는 메타인지'의 중요성 체득" |

### 실전 자소서 예시 — 항목별

**항목 1: 자기주도 학습 경험 (800자 이내)**

> [좋은 예시]
> "중학교 2학년 여름, 수학 이차방정식 단원에서 판별식 문제를 세 번 연속 틀렸습니다. 학원 선생님은 '공식을 외워라'고 했지만, 왜 D=b^2-4ac가 그래프의 교점 수를 결정하는지 이해가 되지 않았습니다.
> 
> 그래서 학원을 쉬고 스스로 탐구하기로 결심했습니다. 먼저 교과서의 설명을 다시 읽었지만 부족했습니다. ChatGPT에 '판별식의 기하학적 의미'를 질문했으나, 답변이 일반적이어서 만족스럽지 않았습니다. 이때 깨달은 것이 있습니다 — AI는 '설명'은 해주지만, '이해'는 내가 직접 해야 한다는 것입니다.
> 
> 직접 y=x^2+bx+c에서 b, c 값을 바꿔가며 30개의 그래프를 그렸습니다. D>0, D=0, D<0일 때 포물선이 x축과 만나는 방식이 달라지는 것을 눈으로 확인하면서 비로소 이해했습니다. 이 경험으로 '모르는 것을 정확히 찾는 메타인지 능력'이 학습의 핵심임을 깨달았습니다."

### 자소서 작성 시 피해야 할 10가지 실수

| 번호 | 실수 | 올바른 방향 |
|------|------|-----------|
| 1 | 학원 경험을 자기주도 학습으로 포장 | 학원이 아닌 본인 주도 경험 서술 |

| 4 | 다른 학교 자소서 복사·붙여넣기 | 해당 학교 프로그램과 구체적 연결 |

| 6 | 지원 동기에 학교 홈페이지 문구 나열 | 본인이 왜 이 학교에 가고 싶은지 |
