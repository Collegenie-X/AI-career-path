# 특수학교 면접 대비 질문집

사관학교/경찰대/영재고 면접에서 공통으로 자주 나오는 질문과 답변 틀입니다.

## 핵심 질문 6개

1. 왜 이 학교(기관)를 선택했나요?
2. 본인의 강점이 조직에 어떤 기여를 하나요?
3. 실패 경험과 극복 과정을 설명해 주세요.
4. 최근 관심 사회 이슈와 본인 의견은?
5. 협업 갈등 상황에서 어떻게 대응했나요?
6. 10년 후 어떤 모습으로 성장하고 싶나요?

## 답변 프레임

| 단계 | 설명 | 예시 문장 |
| --- | --- | --- |
| 배경 | 경험 맥락 제시 | "중2 때 ... 경험이 있었습니다." |
| 행동 | 내가 한 행동 | "저는 ... 방식으로 실행했습니다." |
| 결과 | 숫자/성과 제시 | "결과적으로 ... 성과를 냈습니다." |
| 확장 | 지원동기 연결 | "그래서 귀교에서 ... 기여하고 싶습니다." |

## 면접 당일 체크

- 60초 자기소개 버전/20초 요약 버전 모두 준비
- 질문 의도를 먼저 파악하고 핵심 2문장으로 시작
- 과장 대신 사실 기반 경험만 제시
