### 2-3. AI 시대의 새로운 문제

> **"AI가 다 해주면 내가 할 게 없다"** — 이것이 새로운 불안이다.

| 기존 불안 | AI 시대의 불안 | DreamPath 해결 |
|----------|-------------|--------------|
| "결과물을 어떻게 만들지?" | "AI가 다 만드는데 내가 뭘 해야 하지?" | 기획하는 과정이 진짜 실력 |
| "포트폴리오가 없다" | "AI가 만든 것을 내 것이라 할 수 있나?" | 설계 과정이 나만의 포트폴리오 |
| "방향을 모르겠다" | "AI한테 물어봐도 내 상황에 맞지 않다" | 나만의 커리어 패스로 좁혀가기 |

## 3. 솔루션 (Solution)

### 3-1. DreamPath의 핵심 가치

```
기존 방식:
  계획 세우기 (어렵고 막막) ──── 실행 ──── 결과물 제출
       ↑ 여기서 대부분 포기

DreamPath:
  클릭 → 패스 선택 ──── 나만의 것으로 좁히기 ──── 과정이 자동으로 기록됨
       ↑ 누구나 즉시 시작 가능
```

### 3-2. 메인 기능: 클릭으로 커리어 패스 만들기

```mermaid
graph LR
    subgraph "STEP 1: 클릭으로 시작 (메인)"
        A["200개 직업 패스 탐색"] --> B["클릭 → 상세 정보 확인"]
        B --> C["이 패스 사용하기 클릭"]
        C --> D["즉시 내 커리어 패스 생성"]
    end
    subgraph "STEP 2: 나만의 것으로 좁히기 (메인)"
        D --> E["학년·관심사 필터링"]
        E --> F["범위·주제 좁히기"]
        F --> G["나만의 커리어 패스 완성"]
    end
    subgraph "STEP 3: 과정이 기록된다 (서브)"
        G --> H["AI 멘토링 실행 지원"]
        H --> I["런치패드 모임 참가"]
        I --> J["드림북 자동 완성"]
    end
```

### 3-3. 나만의 계획으로 좁혀가기

커리어 패스는 처음 선택한 그대로 끝나지 않습니다.  
점점 나만의 범위와 주제로 발전시켜 나갑니다.

```
처음: 200개 패스 중 "의사 커리어 패스" 선택
    ↓
좁히기 1: 중2~고3 전체 → 고1~고3 집중
    ↓
좁히기 2: 전체 활동 → 과학 탐구 + 의료 봉사 특화
    ↓
좁히기 3: 일반 의사 → 소아과 전문의 목표로 구체화
    ↓
결국: 나만의 고유한 커리어 패스 완성
     (결국 커리어 패스 과정만 남는다)
```

### 3-4. DreamPath vs 기존 방식 비교

| 항목 | 입시 컨설턴트 | 유학원 | 커리어넷 | **DreamPath** |
|------|-------------|--------|---------|---------------|
| **커리어 패스 생성** | 수동, 300만원 | 없음 | 없음 | ✅ **클릭 한 번** |
| **나만의 계획 커스텀** | 1회성 | 없음 | 없음 | ✅ **무제한 좁히기** |
| **과정 자동 기록** | 없음 | 없음 | 없음 | ✅ **드림북 자동 완성** |
| **실행 지원** | 분기 1회 체크 | 서류 대행 | 없음 | ✅ AI 멘토링 (서브) |
| **비용** | 300만원+ | 1,000만원+ | 무료 | **무료~월 19,900원** |
| **접근성** | 서울/수도권 | 서울/수도권 | 전국 | **전국 + 해외** |
