# 커리어 패스 UI 개선 사항

## 📋 변경 사항 요약

### 1. 다이얼로그 UI 개선 ✨
- **유튜브 스타일 적용**: 깔끔하고 직관적인 인터페이스
- **타임라인 뷰로 변경**: 학년별 시간 순서대로 상세 정보 표시
- **통일성 유지**: 기존 타임라인 컴포넌트와 동일한 스타일 적용

### 2. 탭 기반 네비게이션 🎯
- **상세정보 탭**: 커리어 패스의 학년별 상세 내용
- **댓글 탭**: 유튜브 스타일의 댓글 시스템
- 탭 전환으로 컨텐츠와 댓글을 명확하게 분리

### 3. 공유 기능 제거 🔒
- 탐색 페이지에서 공유 기능 제거
- 공유는 내 컨텐츠(타임라인)에만 존재
- 남의 컨텐츠를 공유하는 것은 의미 없음

## 🎨 UI 개선 상세

### 헤더 영역
```
┌─────────────────────────────────────┐
│  🩺  의사 커리어 패스          ✕    │
│  🔬 탐구 왕국 · 3개 학년            │
└─────────────────────────────────────┘
```
- 직업 이모지 + 제목
- 왕국 정보 + 학년 수
- 닫기 버튼

### 액션 바 (YouTube 스타일)
```
┌─────────────────────────────────────┐
│  ❤️ 342  🔖     👥 128명 사용      │
└─────────────────────────────────────┘
```
- **좋아요**: 하트 아이콘 + 숫자
- **저장**: 북마크 아이콘
- **사용자 수**: 우측 정렬

### 탭 바
```
┌─────────────────────────────────────┐
│  상세정보  │  댓글 (2)              │
│  ━━━━━━━                            │
└─────────────────────────────────────┘
```
- 활성 탭: 컬러 텍스트 + 하단 바
- 비활성 탭: 회색 텍스트
- 댓글 수 표시

## 💬 댓글 탭 (YouTube 스타일)

### 댓글 입력 영역
```
┌─────────────────────────────────────┐
│  😊  댓글 추가...          [댓글]   │
└─────────────────────────────────────┘
```
- 아바타 이모지
- 인라인 입력창 (하단 보더만)
- 텍스트 입력 시 "댓글" 버튼 표시
- Enter 키로 빠른 작성

### 댓글 리스트
```
┌─────────────────────────────────────┐
│  👨‍🎓  김진로  2일 전                 │
│      정말 체계적인 패스네요!          │
│      👍 12                           │
├─────────────────────────────────────┤
│  👩‍🔬  이미래  5일 전                 │
│      KBO 준비 일정이 구체적으로...    │
│      👍 8                            │
└─────────────────────────────────────┘
```

### 특징
- ✅ 아바타 이모지
- ✅ 사용자 이름 + 타임스탬프
- ✅ 댓글 내용
- ✅ 좋아요 버튼 (ThumbsUp 아이콘)
- ✅ 좋아요 토글 (클릭 시 색상 변경 + 카운트 증가/감소)

## 🔄 인터랙션

### 좋아요 (Like)
- **클릭 전**: ♡ 342 (흰색)
- **클릭 후**: ❤️ 343 (빨간색, 채워짐)
- 실시간 카운트 업데이트
- 배경색 변경

### 저장 (Bookmark)
- **클릭 전**: 🔖 (흰색)
- **클릭 후**: ✅ (노란색)
- 배경색 변경

### 탭 전환
- 부드러운 전환 애니메이션
- 활성 탭 하단 바 표시
- 컬러 변경 (템플릿 색상 적용)

### 댓글 좋아요
- **클릭 전**: 👍 (회색)
- **클릭 후**: 👍 (템플릿 색상, 채워짐)
- 숫자 증가/감소

- ✅ 유튜브 스타일로 친숙함
- ✅ 아이콘과 색상으로 명확한 피드백

## 📝 사용 예시

### 1. 상세정보 확인
1. 카드 클릭 → 다이얼로그 오픈
2. 상세정보 탭에서 학년별 계획 확인
3. 타임라인으로 시간 순서 파악
4. 좋아요/저장 버튼 클릭

### 2. 댓글 작성
1. 댓글 탭 클릭
2. 입력창에 댓글 작성
3. Enter 또는 "댓글" 버튼 클릭
4. 실시간으로 댓글 추가됨

### 3. 패스 사용
1. 모든 정보 확인 후
2. 하단 "이 패스 사용하기" 버튼 클릭
3. 자동으로 빌더로 이동
