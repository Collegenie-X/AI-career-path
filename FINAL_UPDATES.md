## 🎯 핵심 방향

### "클릭 한 번으로 나만의 커리어 패스를 만든다"

> **AI 시대의 핵심 통찰**:  
> AI는 결과를 만드는 데 점점 더 유리해진다.  
> 그렇다면 우리가 집중해야 할 것은 **무엇을 만들지 기획하는 과정**이다.  
> 커리어 패스를 설계하고, 좁히고, 발전시키는 과정 자체가 학생의 진짜 역량이다.  
> **결국 커리어 패스 과정만 남는다.**

## 🎯 과정 중심 철학의 실제 적용

### 커리어 패스 설계 과정이 드림북이 된다

```
기존 방식:
  계획 세우기 (어렵고 막막)
      ↓ 여기서 대부분 포기
  실행
      ↓
  결과물 제출

DreamPath:
  클릭 → 패스 선택 (즉시 시작)
      ↓
  나만의 것으로 좁히기 (과정)
      ↓
  설계 과정이 자동으로 드림북에 기록됨 (결과)
```

### 나만의 계획으로 좁혀가는 과정

```
처음: 200개 패스 중 "의사 커리어 패스" 선택
    ↓ 좁히기 1
중2~고3 전체 → 고1~고3 집중
    ↓ 좁히기 2
전체 활동 → 과학 탐구 + 의료 봉사 특화
    ↓ 좁히기 3
일반 의사 → 소아과 전문의 목표로 구체화
    ↓
결국: 나만의 고유한 커리어 패스 완성
     (결국 커리어 패스 과정만 남는다)
```

## 📱 반응형 디자인

### 다이얼로그 너비 조정
```css
/* Before */
w-full max-h-[90vh] rounded-t-3xl

/* After */
w-full max-w-2xl max-h-[90vh] rounded-3xl
```

### 결과
- 모바일: 전체 너비 사용
- 태블릿/데스크톱: 최대 2xl (672px) 제한
- 가운데 정렬로 깔끔한 레이아웃
- 모든 모서리 라운드 (rounded-3xl)
