## 🎯 핵심 방향

### "클릭 한 번으로 나만의 커리어 패스를 만든다"

> **AI 시대의 핵심 통찰**:  
> AI는 결과를 만드는 데 점점 더 유리해진다.  
> 그렇다면 우리가 집중해야 할 것은 **무엇을 만들지 기획하는 과정**이다.  
> 커리어 패스를 설계하고, 좁히고, 발전시키는 과정 자체가 학생의 진짜 역량이다.  
> **결국 커리어 패스 과정만 남는다.**

### 용어 및 운영 기준 업데이트 (앱 구조 반영)

- 사용자 명칭은 `드림 실행`으로 통일
- 현재 코드 구조는 안정성을 위해 `dreammate` 경로/상태 키 유지
- 가이드는 `커리어 패스` + `드림 실행(피드·탐색)` 템플릿 중심으로 운영
- 자료실은 `드림 경험` 영역에서 통합 정보 가이드를 제공하는 별도 축으로 운영

## ✅ 완료된 업데이트

### 2026-03-17 업데이트

#### 1. AI 시대 기획·계획의 중요성 강조 (Readme.md 상단 추가)

**핵심 메시지**
```
AI 시대의 역량 구조:
- AI가 잘하는 것: 코드 작성, 문서 작성, 디자인 생성, 데이터 분석 (실행)
- AI가 못하는 것: 나만의 목표 설정, 경험 기반 기획, 실행 계획 수립 (기획)

→ 기획·계획만 있다면 AI로 쉽게 결과 생성 가능
→ 기획·계획이 없다면 AI도 방향을 못 잡음
```

**Mermaid 다이어그램 추가**
- AI 역량 구조 시각화 (AI가 잘하는 것 vs 못하는 것)
- 기획·계획의 중요성 강조
- DreamPath 역할: 기획·계획 수립 지원 → AI로 실행 가속화

**AI 시대 진로 준비 공식**
```
나만의 기획·계획 (DreamPath) + AI 실행 도구 (ChatGPT, Cursor 등) = 빠른 결과 생성
```

#### 2. 일반 학생 현실 문제점 도식화

**문제 구조 Mermaid 다이어그램 추가**

1. **기존 진로 준비 시스템의 문제점**
   - 정보 비대칭 문제
   - 고비용 컨설팅 의존 (고입 300만원, 생기부 월 50~100만원, 논문 첨삭 30~50만원)
   - 수능·내신 편중 준비
   - 결과: 가정·학교 여건에 따른 기회 격차 확대

2. **일반 학생 현실: 컨설팅 접근 불가 비율**
   - 전체 고등학생 약 130만명
   - 입시 컨설팅 이용: 약 5~8% (고소득층·수도권 중심)
   - **학교 진로 상담 의존: 약 92~95%**
     - 진로 교사 현실: 학생 300명당 교사 1명, 학기당 1인 15분 상담
     - 학생 딜레마: 학교는 탐구보고서·동아리·생기부 활동 요구
     - 실제 현실: 수능·내신 준비에 바쁨 → 활동 주제 선정 방법 모름 → 형식적 참여로 끝남

3. **DreamPath 해결 방식**
   - 공개형 진로 정보 (고입 30개 학교, 대입 활동·논문, 취업 200개 직업)
   - 실행 가이드 템플릿 (단계별 즉시 시작 가능)
   - 학생 사례 공유 문화 (선배 경험, 그룹 협업)
   - 결과: 컨설팅 없이도 스스로 진로 설계 가능

**핵심 개선 포인트 표 추가**

| 문제 영역 | 컨설팅 이용 (5~8%) | 일반 학생 현실 (92~95%) | DreamPath 방식 |
|---------|------------------|---------------------|---------------|
| **정보 접근** | 컨설턴트 맞춤 정보 제공 | 학교 단편적 안내 → 혼자 구글링, 체계 없음 | 공개형 통합 정보 DB |
| **실행 지원** | 고가 컨설팅 (300만원~1,000만원) | 진로 교사 15분 상담 → 접근 불가, 스스로 찾아야 함 | 무료 가이드 템플릿 + AI 지원 |
| **준비 방향** | 활동·경험·포트폴리오 체계 설계 | 단편적 검사 후 → 수능·내신만 준비 | 경험·활동·포트폴리오 병행 |
| **협업 문화** | 컨설팅 그룹·멘토링 네트워크 | 개인 경쟁, 정보 고립 | 학생 사례 공유·그룹 협업 |

## 🎯 과정 중심 철학의 실제 적용

### 커리어 패스 설계 과정이 드림북이 된다

```
기존 방식:
  계획 세우기 (어렵고 막막)
      ↓ 여기서 대부분 포기
  실행
      ↓
  결과물 제출

DreamPath:
  클릭 → 패스 선택 (즉시 시작)
      ↓
  나만의 것으로 좁히기 (과정)
      ↓
  설계 과정이 자동으로 드림북에 기록됨 (결과)
```

### 나만의 계획으로 좁혀가는 과정

```
처음: 200개 패스 중 "의사 커리어 패스" 선택
    ↓ 좁히기 1
중2~고3 전체 → 고1~고3 집중
    ↓ 좁히기 2
전체 활동 → 과학 탐구 + 의료 봉사 특화
    ↓ 좁히기 3
일반 의사 → 소아과 전문의 목표로 구체화
    ↓
결국: 나만의 고유한 커리어 패스 완성
     (결국 커리어 패스 과정만 남는다)
```
