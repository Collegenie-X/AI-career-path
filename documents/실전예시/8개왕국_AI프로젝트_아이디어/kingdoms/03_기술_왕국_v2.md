### TECH-01: 학교 출석 NFC 태그 게임

**아이디어 출처**: 아버지(IT기업) + 출석 체크 번거로움  
**벤치마킹**:
- TAGBACK (NFC 분실물) → 출석 버전
- 포켓몬 GO (체크인) → 학교 출석

**유저 시나리오**:
```
아침 등교 시 교실 NFC 태그
→ "출석 완료! 연속 7일 달성"
→ 포인트 +10, 캐릭터 성장
→ 한 달 개근 → 레어 배지
→ 학급 출석률 랭킹 1등
→ 매점 상품권 획득
```

**문제-해결**:
- 문제: 종이 출석부 비효율, 지각 관리 어려움
- 해결: NFC 자동화, 게임 요소로 동기 부여

**필요성**: 출석 관리 시간 교사당 일 10분, 연 30시간

**핵심 기능**:
1. NFC 태그 → 자동 출석 기록
2. 연속 출석 → 캐릭터 레벨업
3. 학급 출석률 랭킹 (경쟁)

**도구**: React Native + Firebase + NFC 태그 (개당 500원)

**수익 모델**:
- 학교 라이선스 (학교당 월 15만원)
- NFC 태그 판매 (개당 1,000원)
- 프리미엄 통계 (월 5만원)

**세특**: "NFC 출석 시스템으로 학급 출석률 92% → 98%, 교사 업무 시간 30% 단축"

**유저 시나리오**:
```

**문제-해결**:

- 해결: AI 자동 정리, 맞춤형 복습

1. 사진 → OCR → 구조화
2. AI 요약 + 퀴즈 생성

**도구**: Flutter + GPT-4V + Firebase + Tesseract OCR

**수익 모델**:

### TECH-03: 학교 Wi-Fi 속도 지도 게임

**벤치마킹**:

```
교실에서 와이파이 속도 측정
→ "3층 복도 10Mbps" 기록
→ 포인트 +5
→ 학교 전체 히트맵 완성
→ 느린 구역 학교에 제보

1. 자동 속도 측정 + 위치 기록
2. 히트맵 시각화
3. 학생 참여 → 포인트 적립

**도구**: Flutter + Firebase + Google Maps API

**수익 모델**:
- 학교 네트워크 컨설팅 (건당 50만원)

**세특**: "와이파이 지도로 학교 네트워크 개선 제안, 3개 구역 증설 반영"
