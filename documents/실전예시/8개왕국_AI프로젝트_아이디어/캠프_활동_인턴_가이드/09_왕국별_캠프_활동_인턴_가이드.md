- **KAIST 과학 캠프** (여름/겨울, 5일)

  - 대상: 고1~고2

  - 비용: 80만원
  - 포트폴리오: 연구 논문 초안
  - 연계 프로젝트: EXP-10 (논문 읽기 RPG)

- **학종 전략**: 전공 연계성 고려

### 2. 활동 선택 우선순위
1. **캠프**: 집중 학습 + 네트워킹 (여름 필수)
2. **동아리**: 지속적 활동 + 세특 기록 (학기 중)
3. **인턴**: 실무 경험 + 포트폴리오 (고2~고3)
4. **대회**: 수상 실적 + 검증 (연중)

### 3. 포트폴리오 관리
- **즉시 기록**: 활동 직후 1주일 내 정리
- **사진/영상**: 모든 활동 시각 자료 확보
- **정량적 성과**: 숫자로 표현 (사용자 수, 수익, 수상)
- **연계성**: 캠프 → 동아리 → 프로젝트 → 대회 스토리

### 4. 학종 서류 작성 시
- **세특**: 활동 과정의 시행착오와 성장
- **자소서**: 동기 → 실행 → 성과 → 배움
