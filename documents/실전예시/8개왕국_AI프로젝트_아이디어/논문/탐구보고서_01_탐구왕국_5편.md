## 보고서 1: 게임화가 과학 학습 동기에 미치는 영향

### 📌 기본 정보

| 항목 | 내용 |
|------|------|
| **연구 주제** | 위치 기반 과학 퀴즈 게임이 고등학생의 과학 학습 동기 및 성취도에 미치는 영향 |
| **연구 기간** | 8주 (2026년 3월 ~ 5월) |
| **연구 대상** | 고등학생 100명 (실험군 50명, 통제군 50명) |
| **연구 방법** | 사전-사후 통제집단 설계 |
| **관련 과목** | 과학, 정보, 수학 |

### 🎯 연구 배경 및 목적

**문제 인식:**
- 전통적 과학 수업에서 학생들의 낮은 참여도와 학습 동기 부족
- 스마트폰 사용이 일상화된 청소년들에게 효과적인 학습 방법 필요

**연구 목적:**
- 게임화 요소가 과학 학습 동기에 미치는 영향 규명
- 위치 기반 학습의 효과성 검증
- 학습 성취도 변화 측정

### 📚 이론적 배경

**주요 이론:**
1. **자기결정성 이론 (Self-Determination Theory)**
   - 내재적 동기 vs 외재적 동기
   - 자율성, 유능성, 관계성

2. **게임화 이론 (Gamification Theory)**
   - 포인트, 배지, 리더보드 (PBL)
   - 즉각적 피드백의 중요성

3. **위치 기반 학습 (Location-Based Learning)**
   - 맥락적 학습의 효과
   - 실제 환경과의 연결

### 🔬 연구 방법

#### 1. 연구 설계
```
실험군 (n=50): 게임 앱 사용 + 일반 수업
통제군 (n=50): 일반 수업만
기간: 8주
측정: 사전/사후 검사
```

#### 2. 게임 앱 구성
- **퀴즈 콘텐츠:** 물리, 화학, 생명과학, 지구과학 (각 50문항)
- **게임 요소:** 
  - 포인트 시스템 (정답당 10점)
  - 레벨 시스템 (5단계)
  - 리더보드 (주간/월간)
  - 배지 (업적 달성 시)
- **위치 기반 기능:** 
  - 학교 내 특정 위치에서만 퀴즈 활성화
  - 과학실, 도서관 등 학습 공간 연계

#### 3. 측정 도구
**학습 동기 척도 (SMTSL)**
- 내재적 동기 (10문항)
- 외재적 동기 (10문항)
- 5점 리커트 척도

**과학 성취도 테스트**
- 객관식 30문항
- 주관식 5문항
- 100점 만점

**게임 참여도 로그**
- 일일 접속 시간
- 문제 풀이 수
- 정답률

#### 4. 자료 분석
- 독립표본 t-검정 (집단 간 비교)
- 대응표본 t-검정 (사전-사후 비교)
- 상관분석 (참여도-성취도)
- SPSS 26.0 사용

### 📊 연구 결과

#### 1. 학습 동기 변화

| 집단 | 사전 (M±SD) | 사후 (M±SD) | 변화량 | t값 | p값 |
|------|-------------|-------------|--------|-----|-----|
| 실험군 | 3.2±0.6 | 4.1±0.5 | +0.9 | 8.45 | <.001*** |
| 통제군 | 3.3±0.5 | 3.4±0.6 | +0.1 | 1.23 | .225 |

**핵심 발견:**
- 실험군의 학습 동기가 28% 향상 (p<.001)
- 통제군은 유의미한 변화 없음
- 특히 내재적 동기가 35% 증가

#### 2. 과학 성취도 변화

| 집단 | 사전 (M±SD) | 사후 (M±SD) | 변화량 | t값 | p값 |
|------|-------------|-------------|--------|-----|-----|
| 실험군 | 68.4±12.3 | 85.2±10.5 | +16.8 | 7.82 | <.001*** |
| 통제군 | 69.1±11.8 | 75.3±12.1 | +6.2 | 3.15 | .002** |

**핵심 발견:**
- 실험군의 성취도가 25% 향상
- 통제군 대비 10.7점 더 높은 향상
- 효과 크기 (Cohen's d) = 0.92 (큰 효과)

#### 3. 게임 참여도와 학습 효과

**상관분석 결과:**
- 일일 접속 시간 ↔ 학습 동기: r=.68 (p<.001)
- 문제 풀이 수 ↔ 성취도: r=.72 (p<.001)
- 정답률 ↔ 자기효능감: r=.65 (p<.001)

**참여 패턴:**
- 평균 일일 접속: 23분
- 주당 문제 풀이: 평균 85문항
- 8주 지속률: 94% (47/50명)

   - 배지 시스템이 성취감 제공

   - 이동 활동이 신체적 참여 유도

3. **지속 가능성**

#### 선행 연구와의 비교

| 연구 | 동기 향상 | 성취도 향상 | 본 연구 |

**차별점:**

1. **교사를 위한 제언**
   - 수업에 게임화 요소 도입 권장

## 보고서 2: AI 기반 반려동물 건강 모니터링 시스템의 진단 정확도 평가

### 📌 기본 정보

| 항목 | 내용 |
|------|------|
| **연구 주제** | 머신러닝 알고리즘을 활용한 반려동물 질병 조기 진단 시스템의 정확도 및 신뢰도 검증 |
| **연구 기간** | 12주 (2026년 1월 ~ 3월) |
| **연구 대상** | 반려동물 증상 데이터 1,000건 |
| **연구 방법** | 비교 연구 (AI 진단 vs 수의사 진단) |
| **관련 과목** | 생명과학, 정보, 수학 |

### 🎯 연구 배경 및 목적

**문제 인식:**
- 반려동물 양육 가구 증가 (전국 1,500만 가구, 2025년 기준)
- 수의사 진료비 부담 (평균 5만원/회)
- 질병 조기 발견의 중요성 (치료 성공률 2배 향상)

**연구 목적:**
- AI 기반 진단 시스템의 정확도 검증
- 수의사 진단과의 일치도 평가
- 조기 진단 효과성 분석

### 📚 이론적 배경

**주요 개념:**

1. **머신러닝 알고리즘**
   - CNN (Convolutional Neural Network): 이미지 분석
   - Random Forest: 증상 데이터 분류
   - SVM (Support Vector Machine): 패턴 인식

2. **수의학적 진단 기준**
   - 임상 증상 (식욕, 활동성, 배설 등)
   - 신체 검사 소견
   - 검사 결과 (혈액, 영상 등)

3. **진단 정확도 지표**
   - 민감도 (Sensitivity): 실제 질병을 올바르게 진단
   - 특이도 (Specificity): 정상을 올바르게 진단
   - 정확도 (Accuracy): 전체 정확한 진단 비율

### 🔬 연구 방법

#### 1. 데이터 수집

**데이터 출처:**
- 협력 동물병원 3곳
- 기간: 2025년 7월 ~ 12월 (6개월)
- 총 1,000건 (질병 700건, 정상 300건)

**데이터 구성:**
```
- 기본 정보: 종, 나이, 성별, 체중
- 증상 데이터: 15개 항목 (식욕, 활동성, 구토, 설사 등)
- 진단 결과: 수의사 확정 진단
- 이미지: 피부, 눈, 배설물 사진 (해당 시)
```

**질병 분류:**
1. 소화기 질환 (30%)
2. 피부 질환 (25%)
3. 호흡기 질환 (20%)
4. 비뇨기 질환 (15%)
5. 기타 (10%)

#### 2. AI 모델 개발

**알고리즘 선택:**
- **CNN 모델**: 이미지 기반 진단 (피부, 눈 등)
- **Random Forest**: 증상 데이터 분류
- **앙상블 모델**: 두 모델 결합

**모델 학습:**
```python
# 데이터 분할
train_data: 70% (700건)
validation_data: 15% (150건)
test_data: 15% (150건)

# 학습 파라미터
epochs: 100
batch_size: 32
learning_rate: 0.001
```

**특징 추출:**
- 증상 데이터: 15개 특징
- 이미지 데이터: CNN으로 자동 추출
- 통합 특징: 25개 차원

#### 3. 평가 방법

**비교 대상:**
- AI 모델 진단
- 수의사 진단 (Gold Standard)
- 보호자 자가 판단

**평가 지표:**
```
정확도 (Accuracy) = (TP + TN) / (TP + TN + FP + FN)
민감도 (Sensitivity) = TP / (TP + FN)
특이도 (Specificity) = TN / (TN + FP)
F1 Score = 2 × (Precision × Recall) / (Precision + Recall)
```

### 📊 연구 결과

#### 1. 전체 진단 정확도

| 모델 | 정확도 | 민감도 | 특이도 | F1 Score |
|------|--------|--------|--------|----------|
| CNN | 82.3% | 79.5% | 87.2% | 0.81 |
| Random Forest | 85.7% | 83.1% | 90.3% | 0.84 |
| 앙상블 | **88.4%** | **86.2%** | **92.1%** | **0.87** |
| 보호자 판단 | 61.2% | 55.3% | 71.4% | 0.59 |

**핵심 발견:**
- 앙상블 모델이 가장 높은 정확도 (88.4%)
- 수의사 진단과 88.4% 일치
- 보호자 판단 대비 27.2%p 높음

#### 2. 질병별 진단 정확도

| 질병 분류 | 정확도 | 민감도 | 특이도 |
|----------|--------|--------|--------|
| 소화기 질환 | 91.2% | 89.3% | 94.1% |
| 피부 질환 | 93.5% | 91.8% | 96.2% |
| 호흡기 질환 | 84.7% | 81.2% | 89.3% |
| 비뇨기 질환 | 82.1% | 78.5% | 87.4% |
| 기타 | 79.3% | 75.2% | 84.6% |

**핵심 발견:**
- 피부 질환 진단이 가장 정확 (93.5%)
- 이미지 기반 진단의 우수성

#### 2. AI 논문 요약 플랫폼 개발

**시스템 구성:**

**AI 요약 기능:**
- GPT-4 기반 자동 요약
- 3단계 요약 제공:
  1. 초록 수준 (200자)
  2. 핵심 요약 (500자)
  3. 상세 요약 (1,000자)

**게임화 요소:**
```
레벨 시스템:
- 논문 읽기 → 경험치 획득
- 퀴즈 정답 → 보너스 경험치
- 레벨업 → 새로운 분야 해금

배지 시스템:
- 첫 논문 읽기
- 10편 완독
- 어려운 논문 도전
- 완벽한 퀴즈 점수

리더보드:
- 주간 읽기 순위
- 이해도 점수 순위
```

**학습 지원 기능:**
- 전문 용어 사전
- 논문 구조 가이드
- 읽기 전략 팁
- 동료 토론 게시판

**논문 큐레이션:**
- 고등학생 수준별 논문 선정
- 5개 분야 (과학, 기술, 사회, 인문, 예술)
- 각 분야 10편 (총 50편)

#### 3. 측정 도구

구성:

- 비판적 이해 (10문항): 평가 및 판단

### 📊 연구 결과

#### 1. 논문 이해도 변화

**전체 이해도:**

| 집단 | 사전 (M±SD) | 사후 (M±SD) | 변화량 | t값 | p값 | Cohen's d |
|------|-------------|-------------|--------|-----|-----|-----------|
| 실험군 | 45.2±8.3 | 70.1±7.2 | +24.9 | 15.82 | <.001*** | 3.19 |
| 통제군 | 46.1±7.9 | 58.3±8.5 | +12.2 | 7.45 | <.001*** | 1.47 |

**핵심 발견:**
- 실험군 55.1% 향상 (p<.001)
- 통제군 26.5% 향상
- 실험군이 통제군 대비 12.7점 더 높음 (p<.001)
- 효과 크기 매우 큼 (d=3.19)

**영역별 이해도:**

| 영역 | 실험군 변화 | 통제군 변화 | 차이 | p값 |
|------|------------|------------|------|-----|
| 사실적 이해 | +8.2점 | +5.1점 | +3.1점 | <.001*** |
| 추론적 이해 | +8.9점 | +4.3점 | +4.6점 | <.001*** |
| 비판적 이해 | +7.8점 | +2.8점 | +5.0점 | <.001*** |

**핵심 발견:**
- 모든 영역에서 실험군이 유의미하게 높음
- 특히 비판적 이해에서 가장 큰 차이

#### 2. 비판적 읽기 능력 변화

| 능력 | 실험군 (사전→사후) | 통제군 (사전→사후) | 차이 | p값 |
|------|-------------------|-------------------|------|-----|
| 분석 능력 | 3.2 → 4.3 (+1.1) | 3.3 → 3.7 (+0.4) | +0.7 | <.001*** |
| 평가 능력 | 3.1 → 4.2 (+1.1) | 3.2 → 3.6 (+0.4) | +0.7 | <.001*** |
| 추론 능력 | 3.3 → 4.4 (+1.1) | 3.4 → 3.8 (+0.4) | +0.7 | <.001*** |

**핵심 발견:**
- 실험군 모든 영역 34.4% 향상
- 통제군 12.1% 향상
- 실험군이 통제군 대비 2.8배 높은 향상

#### 3. 논문 읽기 속도 변화

| 집단 | 사전 WPM | 사후 WPM | 변화 | 이해도 유지 |

| 통제군 | 188 | 215 | +27 (+14.4%) | 83% → 85% |

**핵심 발견:**
- 실험군 읽기 속도 39.5% 향상
- 이해도는 유지 또는 향상
- 효율적 읽기 전략 습득

#### 4. 학술 자신감 변화

**논문 읽기 자신감:**

| 집단 | 사전 | 사후 | 변화 | p값 |

1. **논문 이해도 향상**

   - 스캐폴딩 효과 확인
   - 점진적 독립 읽기 가능

2. **비판적 읽기 능력 발달**
   - 요약과 원문 비교를 통한 분석 능력 향상
   - 퀴즈를 통한 평가 능력 연습
   - 토론을 통한 추론 능력 발달

   - 읽기 전략 습득
   - 핵심 파악 능력 향상
   - 이해도 유지하며 속도 증가

4. **학술 자신감 증가**
   - 성공 경험 축적
   - 자기효능감 향상
   - 학술 활동 참여 의향 증가

1. **인지 부하 이론**
   - AI 요약이 외재적 인지 부하 감소

   - 학습 효율성 향상

2. **스캐폴딩 이론**
   - 3단계 요약이 점진적 지원 제공
   - 독립적 읽기로의 전환 촉진
   - 비계 제거 과정 확인

3. **메타인지 이론**
   - 요약 비교를 통한 이해 점검
   - 자기 모니터링 능력 발달
   - 학습 조절 전략 습득

   - 학술 리터러시 교육에 AI 도구 통합
   - 단계적 읽기 지도
   - 비판적 읽기 활동 설계

3. **정책적 제언**
   - 교육용 AI 도구 개발 지원
   - 학술 리터러시 교육 강화
   - 디지털 리터러시 교육과 통합
