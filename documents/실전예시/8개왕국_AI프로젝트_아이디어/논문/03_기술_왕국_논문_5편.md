# 논문 1: NFC 출석 시스템 효율성 연구

## Efficiency and User Acceptance of Gamified NFC-Based Attendance System in High Schools

**저자:** 김코딩¹, 박시스템²  
**소속:** ¹서울과학고등학교, ²KAIST 전산학부  
**키워드:** NFC, 출석 관리, 게임화, 사용자 수용, 기술 수용 모델

**제출:** IEEE Transactions on Learning Technologies  
**날짜:** 2026년 3월 16일

---

## 초록 (Abstract)

**배경:** 전통적 출석 관리 방식은 시간 소모적이고 오류가 발생하기 쉽다.  
**목적:** 본 연구는 게임화된 NFC 기반 출석 시스템의 효율성 및 사용자 수용도를 평가하고자 하였다.  
**방법:** 3개 학교 6개 학급 180명을 대상으로 8주간 준실험 연구를 수행하였다. 실험집단(3개 학급, 90명)은 NFC 출석 시스템을 사용하였고, 통제집단(3개 학급, 90명)은 기존 방식(호명)을 사용하였다. 출석 처리 시간, 정확도, 출석률, 사용자 만족도, 기술 수용 모델(TAM) 척도를 측정하였다.  
**결과:** NFC 시스템은 출석 처리 시간을 평균 8.2분에서 0.8분으로 90% 단축시켰다(t=15.67, p<.001). 데이터 정확도는 99.8%로 기존 방식(94.2%)보다 유의미하게 높았다(χ²=12.34, p<.001). 실험집단의 출석률은 95.7%로 통제집단(91.3%)보다 4.4%p 높았다(t=3.45, p=.001). 사용자 만족도는 92.3%였으며, TAM 분석 결과 인지된 유용성(β=.68, p<.001)과 인지된 사용 용이성(β=.45, p<.001)이 사용 의도를 유의미하게 예측하였다.  
**결론:** 게임화된 NFC 출석 시스템은 효율적이고 정확하며 사용자 수용도가 높아 학교 현장 도입 가능성이 높다.

## 1. 서론 (Introduction)

### 1.1 연구 배경

**📝 연구 맥락:**
출석 관리는 학교 행정의 핵심 업무이지만, 전통적 방식(호명, 수기 기록)은 시간 소모적이고 오류가 발생하기 쉽다. 교육부(2023)에 따르면, 교사들은 하루 평균 15분을 출석 관리에 사용하며, 이는 연간 약 45시간에 해당한다. 또한 수기 기록의 오류율은 평균 5.8%로, 학생 불이익 및 행정 분쟁의 원인이 되고 있다.

**❗ 연구 문제:**
- 문제 1: 출석 관리의 시간 비효율성 (하루 15분 소요)
- 문제 2: 수기 기록의 오류 발생 (평균 5.8%)
- 문제 3: 학생의 낮은 출석 동기 (지각률 8.7%)

**🎯 연구 필요성:**
- **이론적 필요성:** NFC 기술의 교육 분야 적용 효과 검증
- **실용적 필요성:** 효율적인 출석 관리 시스템 개발
- **사회적 필요성:** 교사 행정 업무 경감 및 교육 시간 확보

---

### 1.2 연구 목적 및 연구 질문

**🔍 연구 목적:**
본 연구의 목적은 게임화된 NFC 기반 출석 시스템이 출석 관리 효율성, 데이터 정확도, 학생 출석률, 사용자 만족도에 미치는 영향을 검증하고, 기술 수용 모델을 통해 사용자 수용 요인을 분석하는 것이다.

**❓ 연구 질문:**
1. RQ1: NFC 출석 시스템이 출석 처리 시간을 단축시키는가?
2. RQ2: NFC 출석 시스템이 데이터 정확도를 향상시키는가?
3. RQ3: NFC 출석 시스템이 학생 출석률을 향상시키는가?
4. RQ4: 사용자 수용도에 영향을 미치는 요인은 무엇인가?

**🎯 연구 가설:**
- H1: NFC 시스템은 기존 방식보다 출석 처리 시간을 80% 이상 단축시킬 것이다.
- H2: NFC 시스템은 기존 방식보다 데이터 정확도가 95% 이상일 것이다.
- H3: NFC 시스템 사용 학급의 출석률이 기존 방식 학급보다 높을 것이다.
- H4: 인지된 유용성과 사용 용이성은 사용 의도를 정적으로 예측할 것이다.

## 2. 문헌 고찰 (Literature Review)

### 2.1 이론적 배경

#### 2.1.1 NFC 기술 (Near Field Communication)

**📚 정의 및 개념:**
NFC는 13.56MHz 주파수를 사용하는 근거리 무선 통신 기술로, 10cm 이내에서 데이터를 교환할 수 있다(Coskun et al., 2013). RFID 기술의 일종이지만, 양방향 통신이 가능하고 보안성이 높다.

**🔧 주요 특징:**
- 통신 거리: 10cm 이내
- 통신 속도: 최대 424 kbps
- 작동 모드: Reader/Writer, Peer-to-Peer, Card Emulation
- 보안: 암호화 지원, 짧은 거리로 도청 어려움

**🔗 교육 분야 적용 연구:**
- Ozdenizci et al.(2015): NFC 기반 출석 시스템 프로토타입
- Sahandi et al.(2013): 대학 출석 관리 NFC 적용 사례
- Benyo et al.(2012): 모바일 NFC 학습 애플리케이션

---

#### 2.1.2 기술 수용 모델 (Technology Acceptance Model, TAM)

**📚 정의 및 개념:**
Davis(1989)의 기술 수용 모델은 사용자의 기술 수용 행동을 설명하는 이론으로, 인지된 유용성(Perceived Usefulness)과 인지된 사용 용이성(Perceived Ease of Use)이 사용 의도를 결정한다고 주장한다.

**🔧 주요 구성 요소:**
- **인지된 유용성(PU):** 기술이 성과를 향상시킬 것이라는 믿음
- **인지된 사용 용이성(PEOU):** 기술을 쉽게 사용할 수 있다는 믿음
- **사용 의도(BI):** 기술을 사용하려는 의도
- **실제 사용(AU):** 실제 기술 사용 행동

**모델 구조:**
```
외부 변수 → PEOU → PU → 사용 의도 → 실제 사용
              ↓
```

**🔗 관련 연구:**
- Venkatesh & Davis(2000): TAM2 확장 모델
- Venkatesh et al.(2003): UTAUT 통합 모델
- King & He(2006): TAM 메타분석 (88편 논문)

---

#### 2.1.3 게임화와 행동 변화

**📚 정의 및 개념:**
게임화는 게임 요소를 비게임 맥락에 적용하여 사용자 참여와 동기를 향상시키는 전략이다(Deterding et al., 2011).

**🔧 출석 관리에 적용 가능한 게임 요소:**
- 포인트: 출석당 포인트 적립
- 배지: 연속 출석 배지 (7일, 30일, 100일)
- 리더보드: 학급별 출석률 순위
- 챌린지: 학급 출석률 목표 달성

**🔗 관련 연구:**
- Hamari et al.(2014): 게임화가 행동 변화에 미치는 영향
- Seaborn & Fels(2015): 게임화 효과 체계적 고찰
- Landers et al.(2017): 게임화 설계 원칙

## 3. 연구 방법 (Methodology)

### 3.1 연구 설계

**🎯 연구 유형:**
- 연구 접근: 양적 연구
- 연구 설계: 사전-사후 통제집단 준실험 설계
- 시간적 차원: 종단 연구 (8주)

**📐 연구 모형:**

```
실험집단 (NFC 시스템):
학급 1,2,3 (n=90) → 8주 사용 → 효율성, 정확도, 출석률, 만족도 측정

통제집단 (기존 방식):
학급 4,5,6 (n=90) → 8주 사용 → 효율성, 정확도, 출석률, 만족도 측정
```

**🔧 연구 절차:**
1. 1주차: 시스템 설치 및 교육 (실험집단)
2. 2-9주차: 데이터 수집 (매일 출석 기록)
3. 10주차: 사후 설문 및 인터뷰
4. 11-12주차: 데이터 분석

### 3.2 NFC 출석 시스템 설계

**🔧 시스템 구성:**

**하드웨어:**
- NFC 태그: NTAG216 (각 교실 출입구 부착)

- 데이터베이스: PostgreSQL 14

**소프트웨어:**
- 모바일 앱: React Native (iOS/Android)
- 백엔드: Node.js + Express
- 프론트엔드 (교사용): Next.js
- 인증: JWT 토큰

**게임화 요소:**
- 포인트: 출석당 10점, 지각 5점
- 배지: 연속 출석 (7일, 30일, 60일 배지)
- 리더보드: 학급별 출석률 순위 (주간, 월간)
- 챌린지: 학급 출석률 95% 달성 시 보상

**보안 기능:**
- 위치 검증: GPS 기반 학교 내 위치 확인
- 시간 제한: 수업 시작 10분 전부터 5분 후까지만 인정
- 중복 방지: 1일 1회 제한
- 대리 출석 방지: 랜덤 얼굴 인증 (주 1회)

#### 3.3.3 종속변수 2: 데이터 정확도

**📝 조작적 정의:**
출석 기록의 정확성 (실제 출석 여부와 기록 일치도)

- 오류 유형 분류:
  1. 출석인데 결석 기록 (False Negative)

### 4.2 가설 검증 결과

**📝 해석:**
하루 7.4분 절약은 연간 약 44시간(학기당 22시간)에 해당하며, 이는 교사가 교육 활동에 더 집중할 수 있는 시간을 제공한다.

### 5.2 연구 결과의 해석

#### 5.2.1 발견 1에 대한 논의: 시간 효율성

**📝 결과 해석:**

- 교육 시간 확보: 하루 7분 = 주 35분 = 학기 14시간
- 수업 집중도 향상: 출석 후 바로 수업 시작
- 교사 스트레스 감소: 반복적 행정 업무 자동화

**🔗 선행 연구와의 비교:**
- **일치하는 연구:** Ozdenizci et al.(2015)의 85% 단축과 유사하며, 본 연구의 90% 단축은 게임화 요소로 인한 학생 협조 증가가 추가 기여한 것으로 해석됨.

**🎯 이론적 함의:**
자동화 기술이 단순 반복 업무를 효과적으로 대체할 수 있음을 입증하였다.

**💡 실무적 함의:**
학교는 NFC 출석 시스템을 도입하여 교사의 행정 업무를 경감하고, 교육 활동에 더 많은 시간을 투자할 수 있다.

---

#### 5.2.2 발견 2에 대한 논의: 데이터 정확도

**📝 결과 해석:**
99.8%의 높은 정확도는 자동화된 데이터 수집이 인간 오류를 제거한 결과이다. 기존 방식의 5.8% 오류율은 학생 불이익 및 학부모 민원의 원인이 되었으나, NFC 시스템은 이를 거의 제로로 만들었다.

**정확도 향상의 가치:**
- 학생 불이익 방지: 잘못된 결석 기록으로 인한 피해 제거
- 행정 분쟁 감소: 출석 관련 민원 90% 감소
- 데이터 신뢰성: 정확한 출석 통계로 정책 수립 가능

**🎯 이론적 함의:**
자동화 시스템이 데이터 품질을 크게 향상시킬 수 있음을 입증하였다.

**💡 실무적 함의:**
정확한 출석 데이터는 학생 상담, 학업 지원, 정책 수립의 기초 자료로 활용될 수 있다.

---

#### 5.2.3 발견 3에 대한 논의: 출석률 향상

**📝 결과 해석:**
4.4%p의 출석률 향상은 게임화 요소가 출석 동기를 증진시킨 결과로 해석된다. 특히 연속 출석 배지와 학급 챌린지가 효과적이었다.

**게임화 메커니즘:**
- 즉각적 보상: 출석 시 즉시 포인트 획득

- 집단 목표: 학급 챌린지로 소속감

**🔗 선행 연구와의 비교:**
- **새로운 발견:** 기존 NFC 출석 연구는 효율성에만 초점을 맞췄으나, 본 연구는 게임화를 통해 출석률까지 향상시킬 수 있음을 발견하였다.

**🎯 이론적 함의:**

## 초록 (Abstract)

본 연구는 다중 제약 조건을 고려한 유전 알고리즘 기반 학교 시간표 자동 생성 시스템을 개발하고 성능을 평가하였다. 3개 학교의 실제 시간표 데이터를 사용하여 유전 알고리즘(GA), 시뮬레이티드 어닐링(SA), 타부 서치(TS)의 성능을 비교하였다. 15개의 하드 제약(필수)과 10개의 소프트 제약(선호)을 정의하였다. 결과, GA는 하드 제약 만족도 100%, 소프트 제약 만족도 94.7%를 달성하였으며, SA(92.3%)와 TS(89.1%)보다 우수하였다. 시간표 생성 시간은 평균 47분으로, 수동 작성(평균 2주)대비 95% 단축되었다. 교사 만족도는 87.3%였으며, 특히 '공강 시간 배치'와 '연속 수업 방지' 항목에서 높은 만족도를 보였다. 유전 알고리즘은 학교 시간표 최적화에 효과적인 방법임이 입증되었다.
