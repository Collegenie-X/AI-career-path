## 1. 2028 대입 개편 핵심 변화

### 1-1. 무엇이 달라지는가?

| 구분 | 기존 (2027 이전) | 변경 (2028~) |
|------|-----------------|-------------|
| 내신 등급 | 9등급제 | **5등급제** (변별력 약화) |
| 교육과정 | 학년제 | **고교학점제** (과목 선택형) |
| 수능 과목 | 세분화 선택 | **통합형** 재편 |
| 정시 평가 | 수능 점수 100% | **수능 + 학생부 정성평가** |
| 수시 수능 최저 | 일부 적용 | **51.3%로 대폭 확대** |

### 1-2. 3대 결정적 변화

```
┌─────────────────────────────────────────────────┐
│           2028 대입 3대 패러다임 전환              │
├─────────────────────────────────────────────────┤
│                                                 │
│  ① 정시에서도 학생부 정성평가 도입               │
│     └→ 서울대, 고려대, 연세대 등 주요 대학       │
│        수능 점수 + 학교생활 충실도 함께 평가      │
│                                                 │
│  ② 수시 내 수능 최저학력기준 강화                │
│     └→ 5등급제로 내신 변별력 ↓                  │
│     └→ 수시 전형 51.3%가 수능 점수 요구          │
│                                                 │
│  ③ '점수'에서 '과목 설계'로 전환                 │
│     └→ 어떤 과목을 선택했는지가 핵심 변수        │
│     └→ 전공 연계 과목 이수 여부 중요             │
│                                                 │
└─────────────────────────────────────────────────┘
```

### 1-3. 수시 전형별 수능 최저 적용 비율 변화

| 전형 유형 | 2027 (고3) | 2028 (고2) | 변화 |
|-----------|-----------|-----------|------|
| 학생부 종합 전형 | 11.5% | **14.2%** | ↑ |
| 논술 전형 | 91.8% | **98.4%** | ↑ |
| 수시 전체 평균 | 49.7% | **51.3%** | ↑ |

> **핵심 메시지:** 수시를 준비하더라도 수능은 반드시 병행해야 한다!

## 2. 생기부(학교생활기록부) 완전 분석

### 2-1. 생기부 핵심 3대 항목

```
생기부 구조
├── 📘 비교과: 창체 (창의적 체험 활동)
│   ├── 자율활동 ────── 500자
│   ├── 동아리활동 ──── 500자
│   └── 진로활동 ────── 700자
│
├── 📗 교과: 세특 (세부능력 및 특기사항)
│   └── 과목당 ──────── 500자
│
└── 📙 종합: 행특 (행동특성 및 종합의견)
    └── 담임평가 ────── 500자
```

### 2-2. 항목별 상세 분석

| 구분 | 항목 | 글자 수 | 핵심 평가 요소 | 중요도 |
|------|------|---------|---------------|--------|
| 비교과 | 창체 (자율/동아리/진로) | 500/500/700자 | 전공 관심도, 공동체 역량 | ★★★★☆ |
| 교과 | 세부능력 및 특기사항 | 과목당 500자 | 학업 역량, 탐구 태도 | ★★★★★ |
| 종합 | 행동특성 및 종합의견 | 500자 | 인성, 성실성, 변화 과정 | ★★★★☆ |

### 2-3. 각 항목의 역할

```
┌──────────────────────────────────────────────────────┐
│                   창체 (비교과)                        │
│  "이 학생이 어떤 분야에 관심 있고,                     │
│   공동체 안에서 어떤 역할을 하는가?"                    │
├──────────────────────────────────────────────────────┤
│  • 자율활동: 학교 행사·프로그램 참여 및 주도성          │
│  • 동아리: 관심 분야의 지속적 탐구                     │
│  • 진로활동: 진로 탐색 및 구체화 과정                  │
└──────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────┐
│              교과 세특 (가장 중요!)                     │
│  "이 학생이 수업에서 무엇을 배우고,                     │
│   얼마나 깊이 탐구했는가?"                             │
├──────────────────────────────────────────────────────┤
│  • 수업 태도 + 탐구 과정이 핵심                       │
│  • 과목당 500자로 학업 역량 증명                       │
│  • 입학사정관이 가장 중점적으로 보는 항목               │
└──────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────┐
│                행특 (종합 평가)                        │
│  "이 학생의 인성과 성장 가능성은?"                      │
├──────────────────────────────────────────────────────┤
│  • 담임 선생님의 1년간 관찰 종합 평가                  │
│  • 인성, 리더십, 성장 가능성 500자 요약                │
│  • 3년간의 변화 과정이 핵심                            │
└──────────────────────────────────────────────────────┘
```

## 3. 학년별 대입 준비 로드맵

### 3-1. 전체 흐름도

```
[고1: 탐색기]          [고2: 심화기]          [고3: 완성기]
     │                      │                      │
     ▼                      ▼                      ▼
 ┌─────────┐          ┌─────────┐          ┌─────────┐
 │ 다양한   │          │ 관심분야 │          │ 전공     │
 │ 경험     │ ──────→  │ 심화탐구 │ ──────→  │ 역량     │
 │ 쌓기     │          │ 집중     │          │ 증명     │
 └─────────┘          └─────────┘          └─────────┘
     │                      │                      │
 • 교과 탐색           • 세특 심화            • 최종 정리
 • 독서 습관           • 탐구보고서           • 수능 집중
 • 기초 다지기         • 과목 설계            • 자소서 준비
```

### 3-2. 고등학교 1학년 — 탐색과 기초

| 영역 | 세부 활동 | 목표 |
|------|----------|------|
| 교과 | 모든 과목 성실히 이수 | 내신 기반 다지기 |
| 탐구 | 다양한 분야 독서 및 체험 | 관심 분야 발견 |
| 활동 | 동아리 2~3개 탐색 | 적합한 동아리 선택 |
| 수능 | 국·영·수 기초 학습 | 기본 개념 완성 |
| 생기부 | 수업 중 질문·발표 적극 참여 | 세특 기록 소재 확보 |

**1학년 핵심 전략:**
```
Step 1. 수업에서 흥미로운 주제 발견
  │
Step 2. 관련 독서로 배경지식 확장
  │
Step 3. 간단한 탐구 활동 시도
  │
Step 4. 경험을 바탕으로 관심 분야 좁히기
```

### 3-3. 고등학교 2학년 — 심화와 설계

| 영역 | 세부 활동 | 목표 |
|------|----------|------|
| 교과 | 전공 연계 과목 전략적 선택 | 과목 설계 완성 |
| 탐구 | 교과 연계 심화 탐구 프로젝트 | 세특 핵심 소재 확보 |
| 활동 | 핵심 동아리 집중 + 리더십 | 스토리라인 강화 |
| 수능 | 국·영·수 + 탐구과목 본격 대비 | 모의고사 목표 등급 달성 |
| 생기부 | 1학년 경험 → 2학년 심화 연결 | '성장 스토리' 구축 |

**2학년 핵심 전략:**
```
Step 1. 1학년 때 발견한 관심사를 심화 주제로 발전
  │
Step 2. 전공 연계 과목을 전략적으로 선택 (고교학점제 활용)
  │
Step 3. 수업 중 탐구 → 세특에 기록될 심화 활동 수행
  │
Step 4. 수능 최저학력기준 충족을 위한 수능 대비 병행
```

### 3-4. 고등학교 3학년 — 완성과 마무리

| 영역 | 세부 활동 | 목표 |
|------|----------|------|
| 교과 | 전공 심화 과목 마무리 | 교과 이수 체계 완성 |
| 수능 | 6·9월 모의평가 → 실전 대비 | 수능 최저 충족 + 정시 대비 |
| 생기부 | 3년간 스토리라인 최종 점검 | 일관된 성장 서사 완성 |
| 전형 | 수시 6장 카드 전략적 배분 | 상향·적정·안정 조합 |
| 면접 | 생기부 기반 면접 준비 | 활동의 진정성 어필 |

## 4. 수시 vs 정시 전략 비교

### 4-1. 2028 입시에서의 핵심 차이

```
┌─────────────────────┬─────────────────────┐
│       수시 전형      │       정시 전형      │
├─────────────────────┼─────────────────────┤
│ 학생부 + (수능최저)  │ 수능 + 학생부 정성   │
│                     │ 평가 (신설)          │
├─────────────────────┼─────────────────────┤
│ 내신 5등급제로       │ 수능만으로는         │
│ 변별력 ↓            │ 승부 불가            │
├─────────────────────┼─────────────────────┤
│ 51.3% 수능최저 적용  │ 학교생활 충실도      │
│                     │ 함께 평가            │
├─────────────────────┼─────────────────────┤
│   세특이 당락 좌우    │  출결·수업태도 반영   │
└─────────────────────┴─────────────────────┘
```

### 4-2. 정시 파이터의 위험성

```
⚠️ 2028 대입에서 정시만 준비하는 것이 위험한 이유:

  학교 수업 등한시
       │
       ▼
  출결·수업태도 불량 기록
       │
       ▼
  정시 전형에서도 학생부 정성평가 반영
       │
       ▼
  ❌ 치명적 감점 요인 발생!

∴ 결론: 수시든 정시든 '학교생활 충실도'는 필수
```

## 5. 교과 설계 전략 (고교학점제)

### 5-1. 과목 선택 원칙

```
과목 선택 의사결정 트리

                  희망 전공 확인
                      │
          ┌───────────┼───────────┐
          ▼           ▼           ▼
       이공계       인문계       예체능
          │           │           │
     ┌────┴────┐  ┌───┴───┐  ┌───┴───┐
     │ 심화수학 │  │ 사회  │  │ 예술  │
     │ 과학    │  │ 언어  │  │ 실기  │
     │ 정보    │  │ 철학  │  │ 이론  │
     └────┬────┘  └───┬───┘  └───┬───┘
          │           │           │
          ▼           ▼           ▼
     전공 연계성 + 탐구 깊이 = 합격 핵심 변수
```

### 5-2. 계열별 추천 과목 설계

| 희망 계열 | 필수 선택 과목 | 권장 심화 과목 | 연계 포인트 |
|-----------|-------------|-------------|------------|
| 자연/공학 | 미적분, 물리학, 화학 | 기하, 확률과통계, 정보 | 수학·과학 심화 탐구 |
| 의약/생명 | 생명과학, 화학, 수학 | 생명과학II, 화학II | 과학 탐구 보고서 |
| 인문/사회 | 사회문화, 정치와법, 경제 | 세계사, 윤리와사상 | 사회 현상 분석 |
| 상경/경영 | 경제, 수학, 사회문화 | 확률과통계, 정보 | 데이터 기반 분석 |
| 교육 | 교육학, 심리학, 전공과목 | 논리학, 윤리 | 교수법 탐구 |

## 6. 생기부 작성 필승 공식

### 6-1. 세특 '동-활-느' 3박자 원칙

```
세특 작성 공식 (500자 내)

┌─────────────────────────────────────────────┐
│  동기 (왜?)     약 100자                     │
│  ├── 수업 중 어떤 내용에서 호기심이 생겼는가? │
│  └── 왜 이 주제를 더 알고 싶었는가?          │
│                                             │
│  활동 (무엇을?)  약 250자                    │
│  ├── 구체적으로 어떤 탐구를 했는가?           │
│  ├── 어떤 방법론을 사용했는가?               │
│  └── 어떤 결과를 도출했는가?                 │
│                                             │
│  느낀 점 (그래서?) 약 150자                  │
│  ├── 탐구를 통해 무엇을 깨달았는가?          │
│  └── 후속 연구 계획 또는 진로 연결           │
└─────────────────────────────────────────────┘
```

### 6-2. 좋은 세특 vs 나쁜 세특 비교

| 구분 | 나쁜 예시 | 좋은 예시 |
|------|----------|----------|
| 동기 | "수업 시간에 발표를 잘 함" | "DNA 복제 과정에서 오류 수정 메커니즘에 의문을 가짐" |
| 활동 | "관련 자료를 조사함" | "논문 3편을 비교 분석하여 PCR 원리를 실험 설계에 적용" |
| 느낀 점 | "많은 것을 배움" | "유전공학의 윤리적 한계를 인식하고 생명윤리 심화 탐구 계획" |

### 6-3. 독서 활용 전략

```
독서 → 생기부 연결 프로세스

  📖 독서
   │
   ├── ① 수업 주제와 연결할 책 선정
   │
   ├── ② 책 내용을 수업 탐구의 '동기'로 활용
   │
   ├── ③ 독서 내용을 세특 탐구의 '근거'로 인용
   │
   └── ④ 심화 독서 → 후속 탐구 → 성장 스토리 연결

⚡ 포인트: 독서 목록 자체는 대입에 미반영되지만,
   모든 활동의 동기·근거로 활용하면 기록의 질이 3배 ↑
```

### 6-4. 교과 심화 우선의 원칙

```
✅ 올바른 접근                    ❌ 잘못된 접근
                                
수업 중 호기심 발생               무리한 진로 연결
     │                               │
     ▼                               ▼
교과 내용 심화 탐구               억지 스토리 만들기
     │                               │
     ▼                               ▼
자연스러운 전공 연결              입학사정관이 한눈에 파악
     │                               │
     ▼                               ▼
높은 평가 ✅                      신뢰도 하락 ❌
```

## 7. 수능 대비 전략

### 7-1. 수능이 '기본값'인 이유

```
수시 합격 실패 시나리오 분석

  수시 6장 지원
       │
       ├── 1차: 서류 탈락 (생기부 경쟁력 부족)
       ├── 2차: 수능 최저 미충족으로 탈락 ← 가장 흔한 실패 원인!
       └── 3차: 면접 탈락
       
  ∴ 수능 최저를 충족하지 못하면
     아무리 좋은 생기부도 소용없다!
```

### 7-2. 학년별 수능 대비 계획

| 시기 | 국어 | 수학 | 영어 | 탐구 |

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│   ✅ 수능 = 선택이 아닌 '기본값'                     │
│      → 저학년 때부터 수능 대비 병행 필수              │
│                                                     │
│   ✅ 생기부 = '스토리'가 있는 설계                    │
│      → 1학년 탐색 → 2학년 심화 → 3학년 완성          │
│                                                     │
│   ✅ 과목 설계 = 전공 연계 필수                       │
│      → 고교학점제 활용, 전략적 과목 선택              │
│                                                     │
│   ✅ 세특 = '동-활-느' 3박자 원칙                    │
│      → 동기 → 활동 → 느낀 점의 논리적 흐름           │
│                                                     │
│   ✅ 학교생활 충실도 = 수시·정시 모두 필수             │
│      → 정시에서도 학생부 정성평가 반영                │
│                                                     │
└─────────────────────────────────────────────────────┘
```
