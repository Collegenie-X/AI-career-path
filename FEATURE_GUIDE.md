## 🎨 인터랙션 가이드

### 좋아요 (Like)
```
클릭 전: ♡ 342
클릭 후: ❤️ 343 (빨간색)
```
- 실시간 카운트 업데이트
- 색상 변경 (회색 → 빨간색)
- 아이콘 채움 효과

### 즐겨찾기 (Bookmark)
```
클릭 전: 🔖 저장
클릭 후: ✅ 저장 (노란색)
```
- 나중에 다시 보기 위한 북마크
- 색상 변경 (회색 → 노란색)

### 댓글 (Comments)
```
[댓글 입력창] [전송]
```
- 입력 후 Enter 또는 전송 버튼
- 실시간으로 댓글 추가
- 최신 댓글이 위에 표시

### 공유 (Share)
```
공유 버튼 클릭 → 메뉴 펼침 → 옵션 선택
```
- 내부 공유: 즉시 타임라인 게시
- 외부 공유: 클립보드에 링크 복사

## 🎯 사용 시나리오

### 시나리오 1: 학생이 커리어 패스 탐색
1. 탐색 페이지 접속
2. 왕국 필터로 관심 분야 선택
3. 커리어 패스 카드 클릭
4. 상세 정보 확인 (설명, 통계, 학년별 계획)
5. 댓글 읽고 다른 사용자 의견 확인
6. 좋아요 & 북마크
7. "이 패스 사용하기" 클릭
8. 자동으로 빌더로 이동

### 시나리오 2: 선생님이 학생과 공유
1. 타임라인 페이지 접속
2. 커리어 패스 카드 펼치기
3. "공유" 버튼 클릭
4. "외부 공유 (링크 복사)" 선택
5. 복사된 링크를 학생/학부모에게 전달
6. 학생이 링크로 접속하여 확인

### 시나리오 3: 팀원과 내부 공유
1. 타임라인 페이지 접속
2. 커리어 패스 카드 펼치기
3. "공유" 버튼 클릭
4. "내부 공유 (타임라인)" 선택
5. 팀원들의 타임라인에 자동 게시
6. 팀원들이 알림 받고 확인
