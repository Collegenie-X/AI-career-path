# DreamPath 사용자 가이드

## 앱 소개

**DreamPath**는 중·고등학생을 위한 진로 탐색 RPG 앱입니다. 게임처럼 즐기면서 자신의 적성을 발견하고, 다양한 직업을 체험하며, 구체적인 커리어 패스를 설계할 수 있습니다.

---

## 시작하기

### 1. 앱 설치 및 실행

#### 개발 버전 (현재)
- Expo Go 앱 설치 (App Store / Play Store)
- 개발자가 제공한 QR 코드 스캔
- 앱 자동 로드

#### 정식 버전 (추후)
- App Store / Play Store에서 "DreamPath" 검색
- 다운로드 및 설치

### 2. 첫 화면 (스플래시)

앱을 실행하면:
- ✨ DreamPath 로고 애니메이션
- 자동으로 다음 화면으로 이동
  - 처음 사용: 온보딩 화면
  - 재방문: 홈 화면

---

## 온보딩 (처음 사용 시)

### 5단계 소개 슬라이드

1. **나를 발견하다** ✨
   - 적성 검사로 나의 유형 알아보기

2. **직업을 탐험하다** 🧭
   - 8개의 별에서 다양한 직업 만나기

3. **직업을 체험하다** 💼
   - 실제 하루 일과 시뮬레이션

4. **커리어를 설계하다** 🗺️
   - 목표 직업을 위한 구체적인 계획

5. **꿈을 이루다** 🏆
   - 게임처럼 즐기며 진로 탐색

### 프로필 설정

- **닉네임** 입력 (최대 10자)
- **학년** 선택 (초4 ~ 고3)
- "모험 시작하기" 버튼 클릭

## 홈 화면 가이드

홈 화면은 DreamPath의 중심입니다. 모든 주요 기능에 빠르게 접근할 수 있습니다.

### 1. 프로필 & XP 바 (상단)

```
┌─────────────────────────────────┐
│ [Lv.1]  안녕하세요! 👋          ⚙️ │
│         김드림                    │
│         🔥 Lv.1 꿈의 씨앗        │
│                                  │
│ ⚡ 경험치                         │
│ ▓▓▓░░░░░░░ 50 / 300 XP          │
│ 다음 레벨까지 250 XP 남음        │
└─────────────────────────────────┘
```

- **레벨 배지**: 현재 레벨 표시
- **닉네임**: 설정한 닉네임
- **레벨 이름**: 레벨별 타이틀 (꿈의 씨앗 → 드림 마스터)
- **XP 바**: 경험치 진행도
- **설정 버튼**: 프로필, 테마, 데이터 관리

### 2. 내 커리어 패스 섹션

#### 빈 상태 (처음)
```

│ 🚀 내 커리어 패스                │
│                                  │
│ 🎯 → 📚 → 🏆 → 🚀               │
│                                  │
│ 나만의 커리어 패스를              │
│ 만들어봐요!                       │
│                                  │
│ 목표 직업을 정하고 🎯            │
│ 어떤 활동·수상·자격증이           │
│ 필요한지 알아봐요                │
│                                  │

└─────────────────────────────────┘
```

#### 계획이 있을 때
```

│ 🚀 내 커리어 패스      [3개] 전체보기 │
│                                  │
│ ⚡ 2    🏆 1    📜 0              │

│                                  │

```

**기능**:
- 활동/수상/자격증 개수 통계
- 최근 2개 계획 미리보기
- 학년, 목표 월, 소속 별 표시
- 전체보기 버튼 → 커리어 패스 화면

### 3. 내 적성 유형 섹션

#### 검사 전
```
┌─────────────────────────────────┐
│ 🧠 내 적성 유형                  │
│                                  │
│ [🧩]  적성 퀴즈 풀기              │
│       나한테 맞는 직업이          │
│       뭔지 알아봐요!              │
│                                  │
│ ▱▱▱▱▱▱▱▱▱▱ 0%                   │
│ ⚡ 퀴즈 완료하면 +50 XP           │
└─────────────────────────────────┘
```

#### 검사 후
```
┌─────────────────────────────────┐
│ 🧠 내 적성 유형        상세 리포트 › │
│                                  │
│ [🔬] [🤝]                        │
│  1    2                          │
│                                  │
│ 탐구형 + 사회형                  │
│ 분석하며 문제 해결을 즐겨요      │
│                                  │
│ [📋 다시 검사] [🧠 리포트]       │
└─────────────────────────────────┘
```

**RIASEC 유형**:
- 🔧 **현실형 (R)**: 도구로 만들고 고치는 걸 좋아함
- 🔬 **탐구형 (I)**: 분석하며 문제 해결을 즐김
- 🎨 **예술형 (A)**: 상상력으로 창의적 표현
- 🤝 **사회형 (S)**: 사람들을 돕고 협력
- 🚀 **기업형 (E)**: 목표를 세우고 실행
- 📋 **관습형 (C)**: 체계적으로 정리하고 관리

**리포트 보기**:
- 6가지 유형별 점수 그래프
- 상위 2개 유형 상세 설명
- 추천 직업 힌트

```

│ 🎯 나한테 맞는 직업      전체보기 › │
```

**추천 알고리즘**:
- 내 RIASEC 유형과 매칭
- 미래 성장성 (별 5개 = 최고)
- AI 대체 위험도 고려

**별 5개 등급**:
- ★★★★★ (5개): 매우 높은 성장성
- ★★★★☆ (4개): 높은 성장성
- ★★★☆☆ (3개): 보통 성장성

## 주요 기능 상세

### 1. 적성 검사 (RIASEC)

**Holland 적성 검사** 기반 20문항 퀴즈:

- 🔧 **현실형 (R)**: 기계, 도구, 실용적 작업
- 🔬 **탐구형 (I)**: 분석, 연구, 과학적 사고
- 🎨 **예술형 (A)**: 창의성, 예술, 자유로운 표현
- 🤝 **사회형 (S)**: 사람 돕기, 협력, 소통
- 🚀 **기업형 (E)**: 리더십, 목표 달성, 경영
- 📋 **관습형 (C)**: 체계, 정리, 규칙 준수

**결과 확인**:
- 상위 2개 유형 조합 (예: 탐구형 + 사회형)
- 6가지 유형별 점수 그래프
- 각 유형에 맞는 직업 추천

### 2. 직업 탐험 (구현 예정)

#### 8개의 별

1. 🔬 **탐구의 별**: 의사, AI 연구원, 약사
2. 🎨 **창작의 별**: 디자이너, 작가, 영상 크리에이터
3. 💻 **기술의 별**: 개발자, 데이터 과학자, 보안 전문가
4. 🤝 **연결의 별**: 교사, 상담사, 사회복지사
5. 🌿 **자연의 별**: 수의사, 환경 과학자, 조경사
6. 📊 **질서의 별**: 회계사, 변호사, 공무원
7. 📢 **소통의 별**: 마케터, 기자, PD
8. 🏆 **도전의 별**: 기업가, 운동선수, 투자자

#### 직업 정보 레벨

- **L1**: 키워드, 매칭 팁
- **L2**: 하루 일과, 연봉, 전망
- **L3**: 인터뷰, 필요 스킬, 스트레스/자유도/보상
- **L4**: 커리어 마일스톤, 긴급도
- **L5**: 가상 합격자 사례

### 3. 직업 체험 (구현 예정)

#### 하루 시뮬레이션
- 아침/오후/저녁 3개 시나리오
- 각 시나리오마다 2~3개 선택지
- 선택에 따라 스트레스/자유도/보상 점수 변화
- 완료 시 +100 XP

#### 1주일 체험 캠프
- 7일간 미션 (퀴즈, 활동, 성찰, 챌린지, 프로젝트)
- 매일 완료 시 +30 XP
- 전체 완료 시 +200 XP + 배지

### 4. 커리어 패스 빌더 (구현 예정)

#### 목표 직업 선택
- 관심 직업 선택 (예: 의사)
- 자동으로 필요한 활동/수상/자격증 제안

#### 계획 세우기
- **활동**: 과학 실험, 동아리, 봉사 등
- **수상**: 올림피아드, 공모전, 대회 등
- **자격증**: 관련 자격증, 수료증 등

#### 타임라인 뷰
- 초4 ~ 고3까지 학기별 계획
- 예상 비용 계산
- 우선순위 표시

### 5. 런치패드 (구현 예정)

#### Q&A 게시판
- 진로 고민 질문
- 선배들의 답변
- 좋아요/북마크

#### 팀 리크루팅
- 프로젝트 팀원 모집
- 관심 분야별 매칭
- 클루 팀 결성

#### 세션
- 진로 멘토링
- 직업인 특강
- 커리어 워크샵

## 배지 시스템 (구현 예정)

### 배지 카테고리

1. **탐험 배지** 🗺️
   - 첫 별 방문
   - 모든 별 탐험 완료
   - 직업 10개 탐험

2. **시뮬레이션 배지** 🎮
   - 첫 시뮬레이션 완료
   - 시뮬레이션 10개 완료
   - 완벽한 선택 (최고 점수)

3. **커리어 배지** 📋
   - 첫 커리어 패스 작성
   - 커리어 패스 완성
   - 목표 달성

4. **클루 배지** 👥
   - 첫 팀 결성
   - 프로젝트 완료
   - 팀 리더

5. **특별 배지** ⭐
   - 연속 출석 (7일, 30일, 100일)
   - 레벨 마스터 (Lv.6 달성)
   - 전체 직업 탐험 완료

### 배지 효과

- **XP 부스트**: 획득 XP 10~50% 증가
- **기능 해금**: 특정 기능 조기 해금
- **프로필 뱃지**: 프로필에 표시

## 팁 & 트릭

### 빠르게 레벨업하는 법

1. **적성 퀴즈 먼저 완료** (+200 XP)
2. **일일 미션 3개 모두 완료** (+30 XP)
3. **직업 시뮬레이션** (+100 XP)
4. **1주일 캠프 참여** (+200 XP + 배지)

### 효율적인 커리어 패스 작성

1. **목표 직업 먼저 선택**: 자동 제안 활용
2. **학년별로 분산**: 한 학기에 너무 많이 몰리지 않게
3. **비용 확인**: 예산 내에서 계획
4. **우선순위 설정**: 중요한 것부터 먼저

### 적성 검사 활용법

1. **솔직하게 답변**: 남들이 좋아할 답이 아닌 내 마음
2. **첫 느낌으로**: 너무 오래 고민하지 말기
3. **결과 저장**: 리포트 스크린샷 찍어두기
4. **시간 두고 재검사**: 6개월 후 다시 해보기

## 자주 묻는 질문 (FAQ)

### Q1. 적성 검사는 몇 번 할 수 있나요?
**A**: 무제한 가능합니다. 시간이 지나면서 생각이 바뀔 수 있으니 6개월~1년마다 다시 해보는 것을 추천합니다.

### Q2. 커리어 패스는 꼭 따라야 하나요?
**A**: 아니요. 참고용 가이드입니다. 자신의 상황에 맞게 수정하고 조정하세요.

### Q3. 데이터가 삭제되면 어떻게 하나요?
**A**: 현재는 로컬 저장만 지원합니다. 추후 클라우드 백업 기능이 추가될 예정입니다.

### Q4. 직업 정보는 얼마나 정확한가요?
**A**: 실제 직업인 인터뷰, 통계청 데이터, 커리어넷 정보를 기반으로 작성되었습니다. 참고용으로 활용하고, 더 자세한 정보는 전문 진로 상담을 받으세요.

### Q5. 추천 직업이 마음에 안 들어요.
**A**: 추천은 참고용입니다. 직업 체험 탭에서 모든 직업을 자유롭게 탐험할 수 있습니다.

### Q6. 레벨업하면 뭐가 좋나요?
**A**: 새로운 기능이 해금됩니다. 레벨이 높을수록 더 깊이 있는 정보와 도구를 사용할 수 있습니다.

### Q7. 앱을 삭제하면 데이터가 사라지나요?
**A**: 네, 현재는 로컬 저장만 지원하므로 앱 삭제 시 데이터가 사라집니다. 추후 계정 연동 기능이 추가될 예정입니다.

## 앱 사용 플로우 (추천)

### 첫 사용자 (Day 1)

```
1. 온보딩 완료 (프로필 설정)
   ↓
2. 홈 화면 도착
   ↓
3. 적성 퀴즈 풀기 (+200 XP) → Lv.1 → Lv.2
   ↓
4. 추천 직업 3개 확인
   ↓
5. 관심 직업 클릭 → 상세 정보 보기
   ↓
6. 커리어 패스 시작하기
```

### 2일차 이후

```
1. 홈 화면 → 오늘의 미션 확인
   ↓
2. 직업 체험 탭 → 새로운 직업 탐험
   ↓
3. 하루 시뮬레이션 체험 (+100 XP)
   ↓
4. 커리어 패스 탭 → 계획 업데이트
   ↓
5. 런치패드 → Q&A 또는 팀 활동
```

### 장기 사용 (1개월+)

```
1. 여러 직업 체험 완료
   ↓
2. 1주일 캠프 참여
   ↓
3. 클루 팀 결성 (Lv.5 이상)
   ↓
4. 섀도우 프로젝트 진행
   ↓

```

## 학부모/교사 가이드

### 교육적 가치

1. **자기 이해**: RIASEC 검사로 적성 파악
2. **진로 탐색**: 다양한 직업 정보 제공
3. **구체적 계획**: 실행 가능한 커리어 패스
4. **동기 부여**: 게임 요소로 흥미 유지
5. **협업 학습**: 팀 프로젝트 경험

### 활용 방법

#### 가정에서
- 주 1~2회, 자녀와 함께 앱 탐색
- 적성 검사 결과 함께 보며 대화
- 관심 직업에 대해 더 알아보기
- 커리어 패스 함께 계획하기

#### 학교에서
- 진로 수업 보조 자료
- 자유학기제 활동
- 동아리 활동 계획 수립
- 학생부 세특 아이디어 탐색

### 주의사항

- ⚠️ 앱은 **참고용 도구**입니다
- ⚠️ 전문 진로 상담을 대체하지 않습니다
- ⚠️ 학생의 선택을 존중하고 강요하지 마세요
- ⚠️ 직업 정보는 시간에 따라 변할 수 있습니다
