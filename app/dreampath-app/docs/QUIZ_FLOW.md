## 데이터 흐름

```
User Action
    │
    ↓
Quiz Screen (State)
    │
    ├─ currentIndex
    ├─ answers: { 0: 2, 1: 0, ... }
    ├─ showFeedback
    └─ isAnimating
         │
         ↓ (완료 시)
RIASEC Calculator
    │
    ├─ calculateScores()
    ├─ getTopTypes()
    └─ generateKeywords()
         │
         ↓
AsyncStorage
    │
    ├─ riasec_result
    └─ xp_log (+100 XP)
         │
         ↓
Quiz Results Screen
    │
    └─ Display Results
```

## RIASEC 유형 매핑

```
R (현실형) 🔧
├─ 실용적, 활동적
└─ 공학, 제조 직업

I (탐구형) 🔬
├─ 분석적, 호기심
└─ 연구, 의학 직업

A (예술형) 🎨
├─ 창의적, 감성적
└─ 디자인, 예술 직업

S (사회형) 🤝
├─ 따뜻한, 배려심
└─ 교육, 상담 직업

E (기업형) 🚀
├─ 도전적, 리더십
└─ 경영, 기획 직업

C (관습형) 📋
├─ 꼼꼼한, 체계적
└─ 회계, 행정 직업
```

## 점수 계산 로직

```
각 선택지마다 RIASEC 점수 부여
예: { "R": 2, "I": 1 }

30문항 완료 후:
    │
    ↓
총점 계산
R: 15점
I: 22점  ← TOP 1
A: 18점  ← TOP 2
S: 12점
E: 10점
C: 8점
    │
    ↓
TOP 2 유형 선정
[I, A]
    │
    ↓
키워드 생성
["분석적", "호기심 많은", "창의적", "감성적"]
```
