### 3. QuizScreen (퀴즈 진행 화면)

```

│  ← [1/30]                       │ ← Header
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━ │ ← Progress Bar
├─────────────────────────────────┤
│                                 │
│  🧠 성격                         │ ← Zone Badge
│                                 │
│  ┌───────────────────────────┐ │
│  │                           │ │
│  │  친구들과 쉬는 시간이      │ │ ← Question Card
│  │  생겼어요. 나는 주로...    │ │

│  └───────────────────────────┘ │
│                                 │

│  │ Ⓐ 친구들과 시끄럽게...   │   │ ← Choice A
│  └─────────────────────────┘   │
│                                 │
│  ┌─────────────────────────┐   │
│  │ Ⓑ 혼자 책을 읽거나...    │   │ ← Choice B
│  └─────────────────────────┘   │
│                                 │
│  ┌─────────────────────────┐   │
│  │ Ⓒ 작은 그룹으로...       │   │ ← Choice C
│  └─────────────────────────┘   │
│                                 │
│  ┌─────────────────────────┐   │
│  │ Ⓓ 새로운 친구에게...     │   │ ← Choice D
│  └─────────────────────────┘   │

├─────────────────────────────────┤

```

**주요 요소:**
- 헤더 (뒤로가기, 진행률 텍스트, 진행률 바)

- 4개 선택지 (A, B, C, D 라벨)
- 하단 상태바 (완료 수, 색상 점)

**선택 시 애니메이션:**
```
선택 → [+5 XP] 팝업 → 피드백 오버레이
```

## 애니메이션 효과

### 1. 페이드인 (Fade In)
```
초기: opacity: 0, translateY: 20
최종: opacity: 1, translateY: 0
시간: 600ms
```

### 2. XP 팝업
```
표시: 즉시
위치: 화면 중앙 (40%)
지속: 900ms
사라짐: 자동
```

### 3. 피드백 오버레이
```
배경: rgba(0,0,0,0.9)
카드: 중앙 정렬, 스케일 애니메이션
버튼: 사용자 클릭 대기
```

### 4. 진행률 바
```
너비: 0% → 100%
색상: 현재 영역 색상
전환: 부드러운 애니메이션
```

## 반응형 디자인

### Safe Area 적용
- 상단: insets.top
- 하단: insets.bottom
- 노치/홈 인디케이터 고려

### 스크롤 가능 영역
- QuizIntroScreen: 전체 스크롤
- QuizScreen: 질문/선택지 스크롤
- QuizResultsScreen: 전체 스크롤

### 터치 영역
- 버튼: 최소 44x44 포인트
- 선택지: 전체 카드 터치 가능
- hitSlop 적용 (뒤로가기 등)

## 접근성 고려

### 시각적 피드백
- 선택된 항목: 색상 변경
- 버튼 클릭: activeOpacity
- 진행 상태: 진행률 바 + 텍스트

### 텍스트 가독성
- 충분한 대비 (WCAG AA 이상)
- 명확한 폰트 크기
- 적절한 줄 간격

### 사용자 안내
- 명확한 버튼 라벨
- 진행 상황 표시
- 피드백 메시지

이 화면 구성을 통해 사용자는 직관적이고 즐거운 적성 검사 경험을 할 수 있습니다! 🎨
