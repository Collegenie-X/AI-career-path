# 커리어 패스 기능 업데이트

## 📋 개요

DreamPath의 핵심은 **클릭 한 번으로 커리어 패스를 만드는 것**입니다.  
복잡한 설정 없이, 마음에 드는 패스를 클릭하면 즉시 나의 커리어 계획이 됩니다.  
그리고 그 계획은 점점 나만의 범위와 주제로 좁혀지며 발전합니다.

> **핵심 철학**: AI 시대에 결과는 AI가 만드는 데 유리하다.  
> 우리가 집중해야 할 것은 **무엇을 만들지 기획하는 과정**이다.  
> 커리어 패스를 설계하고 좁혀가는 과정 자체가 진짜 실력이고 포트폴리오다.

---

## ✨ 핵심 기능 구조

### 메인 기능: 클릭으로 커리어 패스 만들기

```
탐색 → 클릭 → 사용하기 → 나만의 계획으로 좁히기
```

1. **200개 직업 패스 탐색** — 원하는 직업의 커리어 패스를 찾는다
2. **클릭 → 상세 정보 확인** — 학년별 구성, 활동, 수상, 자격증 미리보기
3. **"이 패스 사용하기" 클릭** — 즉시 내 커리어 패스로 등록
4. **나만의 것으로 좁히기** — 학년, 관심사, 목표에 맞게 커스텀

### 서브 기능: 계획을 바탕으로 실행하기

- AI 프로젝트 멘토링 (실행 지원)
- WBS 프로젝트 관리
- 런치패드 세미나 & 모임 참가

## ✅ 완료된 기능

### 1. 상세 정보 다이얼로그 (CareerPathDetailDialog)

**위치**: `frontend/app/career/components/CareerPathDetailDialog.tsx`

#### 기능:
- 커리어 패스 카드 클릭 시 상세 정보 다이얼로그 표시
- 상세 정보 확인 후 **"이 패스 사용하기"** 버튼으로 원클릭 선택
- 풀스크린 바텀 시트 스타일의 모달 (모바일 최적화)
- 스크롤 가능한 상세 컨텐츠

#### 포함된 정보:
- 상세 설명
- 통계 (사용자 수, 학년 수, 항목 수)
- 항목 구성 (활동, 수상, 자격증, 작품별 개수)
- 학년별 상세 정보 (목표 및 항목 미리보기)
- 태그

### 2. 나만의 타임라인 (VerticalTimelineList)

선택한 커리어 패스를 바탕으로 나만의 계획을 관리합니다.

- 선택한 패스를 타임라인 형태로 시각화
- 학년별 목표와 활동 확인
- 공유 기능 (내부 타임라인 / 외부 링크 복사)
- 수정/삭제 기능

### 3. 좋아요 & 즐겨찾기 기능

- **좋아요**: 커리어 패스에 좋아요 표시 (실시간 카운트 업데이트)
- **즐겨찾기**: 나중에 다시 보기 위한 북마크 기능

### 4. 댓글 기능 (CRUD 완성)

유튜브 스타일의 댓글 시스템:

- 댓글 작성 (Create)
- 댓글 목록 표시 (Read)
- 내 댓글 수정 (Update)
- 내 댓글 삭제 (Delete)
- 다른 사람 댓글 신고 (Report)
- 댓글 좋아요

### 5. 공유 기능

- **외부 공유**: 링크 복사 (고객 공유용)
- **내부 공유**: 타임라인에 공유 (팀원 공유용)
- **신고 기능**: 저작권 침해 또는 부적절한 컨텐츠 신고

## 🎯 나만의 계획으로 좁혀가기

커리어 패스는 처음 선택한 그대로 끝나지 않습니다.  
점점 나만의 범위와 주제로 발전시켜 나갑니다.

```
처음: 200개 패스 중 "의사 커리어 패스" 선택
    ↓
좁히기 1: 중2~고3 전체 → 고1~고3 집중
    ↓
좁히기 2: 전체 활동 → 과학 탐구 + 의료 봉사 특화
    ↓
좁히기 3: 일반 의사 → 소아과 전문의 목표로 구체화
    ↓
결국: 나만의 고유한 커리어 패스 완성
```

**결국에는 커리어 패스 과정만 남는다.**  
계획을 세우고, 좁히고, 발전시키는 과정 자체가 드림북이 된다.

## 📸 주요 화면

### 1. 탐색 페이지
- 카드 리스트 (간단한 정보)
- 카드 클릭 → 상세 다이얼로그

### 2. 상세 다이얼로그
- 헤더 (제목, 이모지, 닫기 버튼)
- 액션 버튼 (좋아요, 북마크, 댓글, 공유)
- 상세 정보 (설명, 통계, 항목 구성, 학년별 상세)
- 댓글 섹션 (CRUD)
- 하단 CTA (**이 패스 사용하기**)

### 3. 나만의 타임라인
- 선택한 커리어 패스 카드 (아코디언)
- 액션 바 (수정, 공유, 삭제)
- 학년별 목표와 활동 시각화
