# 일반고 vs 특목·자사 비교

학교 선택은 입시의 시작입니다. 유형별 현실적인 차이를 비교합니다.

## 핵심 비교표

| 항목 | 일반고 | 특목·자사 |
| --- | --- | --- |
| 내신 경쟁 | 학교별 편차 | 상위권 밀집, 경쟁 강함 |
| 수업 강도 | 중간~높음 | 높음~매우 높음 |
| 프로그램 | 학교별 차이 큼 | 심화·특화 프로그램 많음 |
| 진학 전략 | 수시/정시 선택형 | 투트랙/특화 전략 유리 |
| 적합 학생 | 균형형, 지역 기반 | 고강도 학습 선호, 목표 뚜렷 |

## 선택 질문 5개

1. 고강도 경쟁 환경에서 버틸 체력이 있는가?
2. 내신 중심인가, 수능 중심인가?
3. 통학 시간과 생활 리듬이 유지 가능한가?
4. 원하는 동아리/탐구 인프라가 있는가?
5. 실패 시 대체 루트를 준비했는가?

## 빠른 결론

- **안정 우선**: 일반고 + 내신/학종 정교화
- **상향 도전**: 특목·자사 + 투트랙 운영
