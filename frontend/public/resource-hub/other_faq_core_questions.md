# 자주 묻는 질문 핵심 10선

원문 Q&A에서 실무적으로 많이 묻는 질문만 추렸습니다.

## FAQ 10

1. 재수는 불이익이 있나요? -> 대입 불이익은 거의 없고, 시간관리 이슈가 핵심입니다.
2. 3수 이상은 괜찮나요? -> 목표/데이터 기반 판단이 필수입니다.
3. 검정고시로 명문대 가능한가요? -> 정시/논술 중심으로 가능합니다.
4. 기회균형은 누구나 가능한가요? -> 자격 요건 확인이 먼저입니다.
5. 장애인 특별전형은 어떤 지원이 있나요? -> 대학별 편의제공 제도가 다릅니다.
6. 재수 vs 반수 중 무엇이 유리한가요? -> 집중 가능성은 재수가 유리합니다.
7. 경제적 제약이 크면 어떻게 준비하나요? -> EBS/장학/도서관 루틴 조합이 핵심입니다.
8. 재수 중 알바를 해도 되나요? -> 최소화 권장, 학습 집중 우선입니다.
9. 수능 일정 관리는 어떻게 하나요? -> 접수/모평/본시험 역산 계획이 필수입니다.
10. 대학 진학 후 역전 가능한가요? -> 편입/대학원 루트로 가능합니다.

## 30초 결론

- 본인 상황을 먼저 분류하고,
- 가능한 전형을 좁힌 뒤,
- 실행 루틴을 12주 단위로 운영하세요.
