# 🌟 나를 찾는 여행 — 직업 적성 탐색 검사
> **"나는 어떤 사람일까? 나에게 딱 맞는 직업은 뭘까?"**
> 이 검사는 정답이 없어요. 지금 이 순간의 솔직한 내 모습을 발견하는 시간이에요!

---

## 🗺️ 여행 안내판

```mermaid
flowchart LR
    A([🚀 출발!]) --> B([🧠 ZONE 1<br> 나는 어떤 사람?])
    B --> C([❤️ ZONE 2<br> 뭘 좋아해?])
    C --> D([💪 ZONE 3<br> 뭘 잘 해?])
    D --> E([🌈 ZONE 4<br> 뭐가 중요해?])
    E --> F([📚 ZONE 5<br> 어떻게 배워?])
    F --> G([✨ 점수 정산])
    G --> H([🏆 나의 리포트 완성!])
```

| 항목 | 내용 |
|------|------|
| ⏱️ 소요 시간 | 약 20~30분 (쉬엄쉬엄 해도 괜찮아요!) |
| 📋 문항 수 | 총 75문항 (5개 ZONE × 15문항) |
| 🎯 응답 방법 | 😶전혀 아님(1점) ~ 😄완전 그렇다(5점) |
| 💡 목적 | 나를 더 잘 알고, 내게 맞는 길 찾기 |

> 💬 **이렇게 해보세요!**
> 너무 오래 생각하지 말고 **"음, 나는 어떤 편이지?"** 하고 바로 느끼는 대로 표시하세요.
> 맞고 틀리는 답이 없어요. 지금의 나를 가장 솔직하게 표현하는 게 최고예요! 🙌

| **문항 형식** | "A와 B 중 더 나에게 가까운 것은?" (강제 선택) |
| **결과 보고서** | 4글자 유형 코드 + 유형별 성격 특성 + 의사소통 스타일 + 적합 직업 |
| **저작권** | MBTI는 The Myers-Briggs Company 등록 상표 / 16Personalities는 비공식 무료 유사검사 |
| **장점** | 직관적, 이해하기 쉬움, 대화의 시작점으로 좋음 |
| **한계** | 재검사 시 유형 변동 多, 이분법적 분류의 한계, 과학적 타당도 논쟁 |

**문항 예시 형식:**
```
나는 보통...
(A) 먼저 행동하고 나중에 생각한다
(B) 먼저 생각하고 나중에 행동한다
```

#### 🌊 Big Five / OCEAN (5대 성격 특성)

| 항목 | 내용 |
|------|------|
| **5가지 차원** | O(개방성), C(성실성), E(외향성), A(친화성), N(신경성/정서안정성) |
| **핵심 차이점** | 유형이 아닌 **스펙트럼** 측정 → "외향적이다/아니다"가 아니라 "외향성 72%" |
| **문항 형식** | "나는 새로운 아이디어에 호기심이 많다" → 1(전혀 아님)~5(매우 그렇다) |
| **결과 보고서** | 5개 차원별 백분위 점수 + 레이더 차트 + 하위 요인별 분석 |
| **저작권** | 여러 공개 도구 존재 (IPIP-NEO 등), 교육 목적 무료 활용 가능 |
| **장점** | **심리학계에서 가장 인정받는 성격 모델**, 재검사 안정성 높음, 과학적 근거 풍부 |
| **한계** | 유형 라벨이 없어 직관적 이해가 어려움, 청소년에게 다소 어려울 수 있음 |

**차원별 의미:**

| 차원 | 낮은 점수 | 높은 점수 | 직업 연결 |
|:---:|---------|---------|---------|
| **O** 개방성 | 전통적, 현실적 | 상상력 풍부, 호기심 旺盛 | 높음 → 예술, 연구 / 낮음 → 관리, 행정 |
| **C** 성실성 | 유연, 즉흥적 | 체계적, 목표 지향 | 높음 → 관리직, 의료 / 낮음 → 자유직, 예술 |
| **E** 외향성 | 내성적, 독립적 | 사교적, 에너지 넘침 | 높음 → 영업, 교육 / 낮음 → 연구, 프로그래밍 |
| **A** 친화성 | 경쟁적, 분석적 | 협력적, 공감 능력 높음 | 높음 → 상담, 복지 / 낮음 → 법률, 경영 |
| **N** 신경성 | 정서 안정, 차분 | 감정 기복, 스트레스 민감 | 낮음 → 위기관리, 리더십 / 높음 → 창작, 예술 |

#### 💎 에니어그램 (Enneagram)

| 항목 | 내용 |
|------|------|
| **9가지 유형** | 1(개혁가), 2(조력자), 3(성취자), 4(예술가), 5(탐구자), 6(충성가), 7(열정가), 8(지도자), 9(중재자) |
| **핵심 차이** | 성격이 아닌 **핵심 동기와 두려움**을 탐구 (예: 3유형의 동기="인정받고 싶다", 두려움="무가치해질까 봐") |
| **날개 개념** | 인접 유형의 영향 (예: 4w5 = 예술가+탐구자 성향) |
| **성장/퇴행** | 각 유형의 건강한 상태 → 보통 상태 → 불건강한 상태 경로 제시 |
| **저작권** | 전통적 지혜 기반으로 특정 저작권 없음, 다양한 검사 도구 존재 |
| **장점** | 자기 성찰과 내면 성장에 매우 유용, 대인관계 이해 깊음 |
| **한계** | 과학적 검증 부족, 주관적 해석 여지가 큼 |

#### 📊 DISC 행동유형검사

| 항목 | 내용 |
|------|------|
| **4가지 유형** | D(주도형/Dominance), I(사교형/Influence), S(안정형/Steadiness), C(신중형/Conscientiousness) |
| **문항 형식** | 4개 형용사 중 "가장 나다운 것"과 "가장 안 나다운 것" 선택 |
| **핵심 활용** | 조직 내 **커뮤니케이션 스타일** 파악 → 팀 빌딩, 리더십 개발 |
| **저작권** | DISC 이론 자체는 공개, 검사 도구는 기관별 유료 |
| **장점** | 결과 간명, 조직 환경에서 즉시 적용 가능 |
| **한계** | 성격보다 행동 패턴에 초점, 상황에 따라 결과 변동 |

#### 💪 VIA 성격 강점검사 (Values in Action)

| 항목 | 내용 |
|------|------|
| **6대 미덕** | 지혜, 용기, 인간애, 정의, 절제, 초월 |
| **24가지 강점** | 창의성, 호기심, 판단력, 학구열, 통찰력, 용감함, 인내, 정직, 열정, 사랑, 친절, 사회지능, 협동심, 공정성, 리더십, 용서, 겸손, 신중성, 자기조절, 감상력, 감사, 희망, 유머, 영성 |
| **문항 형식** | "나는 어떤 상황에서도 정직하려고 노력한다" → 1(전혀 나답지 않다)~5(매우 나답다) |
| **결과 보고서** | 24개 강점 순위 + 상위 5개 "대표 강점(Signature Strengths)" |
| **저작권** | **무료** (viacharacter.org), 기본 결과 무료 / 상세 보고서 유료 |
| **장점** | 긍정심리학 기반, **"고칠 것"이 아닌 "살릴 것"에 초점**, 10세부터 검사 가능 |
| **적용** | 강점 활용 진로 설계, 자존감 향상, 팀 구성 |

### 검사 조합 활용 전략 — "3-렌즈 접근법"

> 하나의 검사로 사람을 완전히 이해할 수 없습니다.
> **3가지 렌즈**를 조합하면 더 입체적인 자기 이해가 가능해요!

```mermaid
flowchart TD
    subgraph LENS1["🔍 렌즈 1: 뭘 좋아하나? (흥미)"]
        H[🎯 Holland RIASEC<br>직업 흥미 유형]
    end

    subgraph LENS2["🔍 렌즈 2: 나는 어떤 사람인가? (성격)"]
        M[🧠 MBTI 또는 Big Five<br>성격 유형/특성]
    end

    subgraph LENS3["🔍 렌즈 3: 뭘 잘 하나? (강점)"]
        V[💪 VIA 강점검사<br>24가지 성격 강점]
    end

    H --> R1[관심 분야·직업군 파악]
    M --> R2[소통·의사결정 스타일 파악]
    V --> R3[핵심 강점·가치 파악]

    R1 & R2 & R3 --> FINAL([🏆 입체적 자기 이해<br>+ 직업 적성 종합 판단])
```

| 조합 전략 | 검사 조합 | 알 수 있는 것 | 추천 대상 |
|---------|---------|------------|---------|
| **기본 조합** | Holland + MBTI | 흥미 방향 + 성격 스타일 | 진로 탐색 시작하는 중학생 |
| **심화 조합** | Holland + Big Five + VIA | 흥미 + 성격 스펙트럼 + 강점 | 구체적 진로 설계하는 고등학생 |
| **종합 조합** | 본 검사(5 ZONE) + 커리어넷 + VIA | 통합 자기이해 + 흥미 심화 + 강점 발견 | 깊이 있는 자기 탐색을 원하는 학생 |
